02

Apa itu View?Konsep Dasar

Satu table di Airtable menyimpan satu kumpulan data yang sama, tetapi data itu bisa ditampilkan dengan cara berbeda-beda tanpa mengubah datanya sendiri. Konsep ini setara dengan membuat beberapa Query atau Report berbeda dari satu tabel yang sama di Access — bedanya, di Airtable semua View bisa dibuat dan diganti secara instan lewat panel samping, tanpa menulis query baru.

04

Langkah PraktikMenambahkan 3 View pada Table "Surat Masuk"

  1. Buka table "Surat Masuk" dari Sesi 02.View Grid default sudah otomatis tersedia.
  2. Klik "+ Add view" dan pilih Kanban.Atur "Stack by" berdasarkan field Status (Diproses/Diteruskan/Selesai).
  3. Coba geser satu kartu surat ke kolom status lain.Perhatikan bahwa data di View Grid ikut berubah otomatis.
  4. Tambahkan View baru bertipe Calendar.Atur berdasarkan field "Tanggal Diterima".
  5. Beri nama setiap View secara jelas.Misal: "Semua Data", "Status Proses", "Kalender Surat".
  6. Bandingkan ketiga View dan diskusikan kelebihannya.Catat kapan masing-masing View paling membantu pekerjaan admin.
