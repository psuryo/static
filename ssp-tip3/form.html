Bagian B — 7 Tugas Praktikum
Kerjakan secara berurutan — setiap tugas membangun di atas tugas sebelumnya
1
Form Wizard Tabel: customer ● Mudah
Membuat Form Data Customer

Buat sebuah Form tunggal untuk menampilkan dan mengedit data customer. Form ini akan menjadi Form Utama pada tugas berikutnya.

1
Di panel Navigation, klik kanan tabel customer → pilih Form (bukan Form Wizard) untuk membuat form otomatis.
2
Access akan membuat form dengan semua field. Beralih ke Design View dengan klik ikon di pojok kanan bawah.
3
Atur layout: pindahkan dan sejajarkan label & field agar rapi. Gunakan Arrange → Tabular atau Stacked sesuai preferensi.
4
Ubah Caption form (di Property Sheet → Format → Caption) menjadi: "Form Data Customer".
5
Simpan form dengan nama frm_Customer.

2
Form Wizard Tabel: barang ● Mudah
Membuat Form Data Barang dengan Validasi

Buat form untuk mengelola data barang. Tambahkan Validation Rule untuk memastikan harga dan stok tidak boleh bernilai negatif.

1
Buat form dari tabel barang menggunakan Create → Form. Simpan dengan nama frm_Barang.
2
Masuk ke Design View. Klik field harga, buka Property Sheet (F4), tab Data.
3
Isi properti Validation Rule: >=0 dan Validation Text: "Harga tidak boleh negatif!"
4
Ulangi langkah 2–3 untuk field stok dengan Validation Rule: >=0 dan teks: "Stok tidak boleh negatif!"
5
Ubah field kategori menjadi Combo Box: klik kanan field → Change To → Combo Box. Isi Row Source dengan: "Elektronik";"Aksesoris";"Buku"

Subform Wizard Form & Subform ● Menengah
Membuat Form Transaksi dengan Subform

Ini adalah inti dari modul ini. Buat Form Utama berbasis tabel customer yang di dalamnya terdapat Subform berisi riwayat transaksi customer tersebut.

1
Buat Subform terlebih dahulu. Gunakan Create → Form Wizard, pilih tabel transaksi. Masukkan field: id, barang_id, jumlah, tgl_transaksi. Pilih layout Datasheet. Simpan sebagai subfrm_Transaksi.
2
Buat Form Utama. Buka form frm_Customer yang sudah dibuat di Tugas 1 dalam Design View.
3
Perbesar area Detail form untuk memberi ruang bagi subform. Dari panel Design → Controls, klik ikon Subform/Subreport lalu gambar area di bawah field-field customer.
4
Pada wizard yang muncul, pilih "Use an existing form" dan pilih subfrm_Transaksi. Klik Next.
5
Pada dialog Link Master/Child, pastikan Access sudah mendeteksi otomatis: Link Master Fields: id dan Link Child Fields: customer_id. Klik Finish.
6
Simpan form utama dengan nama frm_TransaksiCustomer.

4
Combo Box Lookup ● Menengah
Mengganti Barang ID dengan Combo Box Lookup

Saat ini subform menampilkan barang_id berupa angka yang tidak informatif. Ubah field ini menjadi Combo Box yang menampilkan nama barang, namun tetap menyimpan ID-nya.

1
Buka subfrm_Transaksi dalam Design View. Klik field barang_id.
2
Klik kanan → Change To → Combo Box.
3
Buka Property Sheet (F4), tab Data. Atur properti berikut:
⚙️ Property Sheet — Combo Box barang_id
PropertiTabNilai
Row Source TypeDataTable/Query
Row SourceDataSELECT id, nama_barang, harga FROM barang ORDER BY nama_barang;
Bound ColumnData1 (kolom pertama = id)
Column CountFormat3
Column WidthsFormat0cm;4cm;2cm
List WidthFormat6cm
Limit To ListDataYes
4
Column Width pertama 0cm menyembunyikan kolom ID. Yang terlihat hanya nama barang dan harganya. Kolom ID tetap tersimpan sebagai nilai.
5
Simpan dan uji: saat mengklik field barang di subform, dropdown harus menampilkan daftar nama barang beserta harganya.
5
Calculated Field Text Box ● Menengah
Menambah Kolom Kalkulasi Total Harga

Tambahkan kolom Total Harga yang menghitung otomatis harga × jumlah di setiap baris transaksi dalam subform, serta Grand Total di footer subform.

1
Buka subfrm_Transaksi dalam Design View. Aktifkan Form Footer: klik kanan di area detail → Form Header/Footer.
2
Di bagian Detail, tambah Text Box baru di sebelah kanan kolom jumlah. Ubah labelnya menjadi "Total Harga".
3
Klik text box tersebut, buka Property Sheet. Di tab Data → Control Source, ketik ekspresi: =[jumlah]*[barang_id].[Column](2)
4
Atur Format: Currency, Enabled: No, Locked: Yes (field ini hanya untuk tampilan, tidak bisa diedit).
5
Di area Form Footer, tambahkan Text Box baru dengan Control Source: =Sum([jumlah]*[barang_id].[Column](2)) dan beri label "Grand Total:".

6
Command Button Navigasi VBA / Macro ● Sulit
Menambahkan Tombol Navigasi dan Aksi

Tambahkan Command Button pada Form Utama untuk mempermudah navigasi dan operasi data tanpa harus menggunakan navigation bar default Access.

1
Buka frm_TransaksiCustomer dalam Design View. Tambahkan area Form Footer.
2
Dari Design → Controls, klik ikon Button dan gambar di Form Footer. Command Button Wizard akan terbuka.
3
Buat 5 tombol dengan pengaturan berikut:
🔧 Daftar Command Button yang Harus Dibuat
Nama TombolKategori WizardAksi Wizard
btn_PrevRecord NavigationGo To Previous Record
btn_NextRecord NavigationGo To Next Record
btn_NewRecord OperationsAdd New Record
btn_SaveRecord OperationsSave Record
btn_DeleteRecord OperationsDelete Record
4
Sembunyikan navigation bar bawaan: buka Property Sheet form (klik area abu-abu di luar form) → tab Format → Navigation Buttons: No.
5
Sejajarkan kelima tombol dengan rapi menggunakan Arrange → Align → Bottom dan Size/Space → Equal Horizontal.
⚠️

Perhatian untuk btn_Delete: Saat tombol hapus diklik, Access akan meminta konfirmasi. Ini perilaku default yang tidak perlu diubah karena melindungi data dari penghapusan tidak sengaja.

7
Search Box Filter VBA / Macro ● Sulit
Menambahkan Fitur Pencarian Customer

Tambahkan kotak pencarian di bagian atas Form Utama agar pengguna bisa mencari customer berdasarkan nama tanpa harus berpindah record satu per satu.

1
Buka frm_TransaksiCustomer dalam Design View. Tambahkan area Form Header: klik kanan → Form Header/Footer.
2
Di Form Header, tambahkan Label bertuliskan "Cari Customer:" dan sebuah Text Box di sebelahnya. Ubah nama text box (Property Sheet → Other → Name) menjadi txt_Cari.
3
Tambah juga Command Button dengan label "Cari", beri nama btn_Cari. Pilih Miscellaneous → Run Macro atau langsung gunakan Event Procedure.
4
Klik tombol btn_Cari → Property Sheet → tab Event → On Click → klik [...] → pilih Code Builder. Ketik kode VBA berikut:
VBA — Event On Click btn_Cari
Private Sub btn_Cari_Click() Dim strCari As String strCari = Me.txt_Cari.Value If strCari = "" Then Me.FilterOn = False ' Hapus filter jika kosong Else Me.Filter = "nama LIKE '*" & strCari & "*'" Me.FilterOn = True End If End Sub
5
Tambahkan juga tombol btn_Reset dengan label "Tampilkan Semua" dan kode: Me.FilterOn = False pada event On Click-nya.
6
Uji: ketik sebagian nama customer (misal "Andi") lalu klik tombol Cari. Form harus memfilter dan hanya menampilkan customer yang namanya mengandung kata tersebut.
✅

Hasil Akhir yang Diharapkan: Form yang lengkap dengan header pencarian, data customer di bagian utama, subform transaksi yang terhubung otomatis, kalkulasi total harga, dan tombol navigasi di footer.
