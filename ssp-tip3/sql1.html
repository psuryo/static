A1. Buat Tabel (DDL)

SQL — DDL
-- 1. Tabel customer
CREATE TABLE customer (
  id          INT PRIMARY KEY,
  nama        VARCHAR(100) NOT NULL,
  email       VARCHAR(100),
  kota        VARCHAR(50),
  tgl_daftar  DATE
);

-- 2. Tabel barang
CREATE TABLE barang (
  id           INT PRIMARY KEY,
  nama_barang  VARCHAR(100) NOT NULL,
  kategori     VARCHAR(50),
  harga        DECIMAL(12,2) NOT NULL,
  stok         INT DEFAULT 0
);

-- 3. Tabel transaksi (mereferensi customer dan barang)
CREATE TABLE transaksi (
  id             INT PRIMARY KEY,
  customer_id    INT NOT NULL,
  barang_id      INT NOT NULL,
  jumlah         INT NOT NULL,
  tgl_transaksi  DATE,
  FOREIGN KEY (customer_id) REFERENCES customer(id),
  FOREIGN KEY (barang_id)   REFERENCES barang(id)
);

Bagian B — 10 Soal SQL
Tulis query SQL yang menjawab setiap pertanyaan di bawah ini
💡

Tuliskan jawaban query SQL kamu pada kolom yang tersedia di bawah setiap soal. Pastikan query-mu dapat dijalankan pada database yang sudah dibuat di Bagian A.

1
SELECT · WHERE
● Mudah
Tampilkan semua data customer yang berasal dari kota Yogyakarta. Urutkan berdasarkan nama secara alfabetis.
Petunjuk: Gunakan WHERE dan ORDER BY.
✏️ Tulis query-mu di sini...
2
SELECT · WHERE · BETWEEN
● Mudah
Tampilkan nama dan harga semua barang dengan harga antara Rp100.000 – Rp500.000, diurutkan dari harga termurah.
Petunjuk: Gunakan BETWEEN dan ORDER BY.
✏️ Tulis query-mu di sini...
3
UPDATE · DML
● Mudah
Stok Flash Disk 64GB (id = 6) habis terjual 10 buah. Perbarui kolom stok menjadi nilai baru yang sudah dikurangi.
Petunjuk: Gunakan UPDATE … SET … WHERE. Hati-hati jangan lupa WHERE!
✏️ Tulis query-mu di sini...
4
SELECT · LIKE
● Mudah
Cari semua barang yang nama-nya mengandung kata “Buku”. Tampilkan nama barang, kategori, dan harganya.
Petunjuk: Gunakan operator LIKE dengan wildcard %.
✏️ Tulis query-mu di sini...
5
INNER JOIN
● Menengah
Tampilkan riwayat transaksi lengkap: tanggal transaksi, nama customer, nama barang, dan jumlah pembelian. Urutkan dari tanggal terbaru.
Petunjuk: Gunakan INNER JOIN untuk menghubungkan ketiga tabel sekaligus.
✏️ Tulis query-mu di sini...
6
JOIN · Kalkulasi Kolom
● Menengah
Hitung total nilai pembelian (harga × jumlah) untuk setiap transaksi. Tampilkan nama customer, nama barang, jumlah, harga satuan, dan total.
Petunjuk: Buat kolom kalkulasi dengan ekspresi b.harga * t.jumlah dan beri alias.
✏️ Tulis query-mu di sini...
7
GROUP BY · SUM · Agregasi
● Menengah
Tampilkan total belanja masing-masing customer (nama customer + jumlah rupiah yang dibelanjakan secara keseluruhan), diurutkan dari yang terbesar.
Petunjuk: Gunakan GROUP BY nama customer dan fungsi SUM.
✏️ Tulis query-mu di sini...
8
LEFT JOIN · IS NULL
● Menengah
Temukan customer yang belum pernah melakukan transaksi sama sekali. Tampilkan nama dan emailnya.
Petunjuk: Gunakan LEFT JOIN lalu filter dengan IS NULL pada kolom dari tabel transaksi.
✏️ Tulis query-mu di sini...
9
GROUP BY · SUM · HAVING
● Sulit
Tampilkan kategori barang beserta jumlah item yang terjual (total satuan, bukan jumlah transaksi). Hanya tampilkan kategori yang total satuannya lebih dari 5.
Petunjuk: Gunakan SUM(jumlah) lalu filter hasil GROUP BY dengan HAVING.
✏️ Tulis query-mu di sini...
10
JOIN · GROUP BY · ORDER BY · LIMIT
● Sulit
Siapa customer paling boros (total belanja tertinggi)? Tampilkan hanya 1 baris: nama customer, kota asal, dan total belanjaannya.
Petunjuk: Kembangkan query Soal 7, tambahkan LIMIT 1 dan ORDER BY DESC.
✏️ Tulis query-mu di sini...
