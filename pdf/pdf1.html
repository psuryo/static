Materi 1

Apa Itu PDF?

Definisi

PDF (Portable Document Format) adalah format berkas yang dikembangkan oleh Adobe pada tahun 1993. Tujuannya satu: agar dokumen terlihat sama persis di semua perangkat, sistem operasi, dan printer — tanpa bergantung pada aplikasi atau font yang digunakan untuk membuatnya.

Bayangkan kamu mengirim laporan ke dosen menggunakan Microsoft Word. Jika dosen membukanya di komputer berbeda, tata letak, font, bahkan isi bisa berubah. PDF menghilangkan masalah ini.

Materi 3

Tiga Cara Membuat PDF

  • 1

    Export / Save As dari Microsoft Word

    Buka dokumen Word → klik File → Save As → pilih format PDF (*.pdf) → klik Save. Cara ini mempertahankan semua tata letak, heading, dan daftar isi yang sudah dibuat di Word.

  • 2

    Download / Export dari Canva

    Di Canva, klik tombol Share (pojok kanan atas) → Download → pilih format PDF Standard (untuk berbagi) atau PDF Print (untuk cetak berkualitas tinggi) → klik Download.

  • 3

    Print to PDF (dari aplikasi apa saja)

    Buka dokumen/halaman web apapun → tekan Ctrl+P (Windows) atau Cmd+P (Mac) → pada kolom Printer, pilih Microsoft Print to PDF atau Save as PDF → klik Print/Save. Cara ini bekerja di hampir semua aplikasi.

💡 Tips

Untuk dokumen resmi yang akan dikirim ke instansi, gunakan cara Export dari Word agar hyperlink dan bookmark tetap berfungsi. Untuk desain visual (poster, brosur), gunakan Canva PDF Print agar warna dan resolusi terjaga.

Latihan & Tugas

Praktikum Sesi 1

🛠 Latihan Kelas (30 menit terakhir)

Setiap mahasiswa menyelesaikan tiga langkah berikut secara mandiri di komputer masing-masing:

  • Buat dokumen singkat di Microsoft Word berisi biodata diri (nama, NIM, prodi, hobi) — lalu simpan sebagai PDF.
  • Buka Canva, pilih template "Kartu Nama" atau "Flyer", isi dengan data diri singkat — lalu download sebagai PDF Standard.
  • Buka halaman web apapun di browser, lalu gunakan Ctrl+P → Save as PDF untuk menyimpan halaman tersebut menjadi PDF.

📌 Tugas Mandiri (dikumpulkan sebelum Sesi 2)

  • Buat sebuah CV sederhana di Microsoft Word (1 halaman), lalu simpan dalam format PDF.
  • Buat poster digital di Canva bertema bebas (minimal 1 gambar, 1 teks utama, 1 teks pendukung) — export sebagai PDF Print.
  • Kumpulkan kedua file PDF tersebut dalam satu folder dengan nama: NIM_Nama_Sesi1 — upload ke LMS/Google Classroom.
