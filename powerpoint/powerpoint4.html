Materi 1

Presenter View: Tampil Percaya Diri

Presenter View adalah tampilan khusus yang hanya terlihat oleh presenter — menampilkan slide saat ini, slide berikutnya, catatan, dan timer, sementara audiens hanya melihat slide biasa di layar utama/proyektor.

Slide saat ini (layar proyektor/audiens)
Slide Berikutnya

Preview slide selanjutnya agar presenter siap bertransisi tanpa kaget

Catatan Pembicara

Poin-poin yang ingin disampaikan — hanya terlihat presenter, tidak muncul di proyektor

Timer

Menghitung waktu presentasi berjalan — membantu manajemen waktu

  • 1

    Tambahkan catatan pembicara terlebih dahulu

    Di tampilan Normal, klik area Notes di bawah slide (View → Notes jika tidak terlihat) → ketik poin-poin yang ingin disampaikan untuk setiap slide.

  • 2

    Aktifkan Presenter View

    Klik tab Slide Show → centang Use Presenter View. Pastikan komputer terhubung ke proyektor/layar kedua agar fitur ini otomatis aktif saat presentasi dimulai.

  • 3

    Mulai presentasi dan gunakan kontrolnya

    Tekan F5 untuk memulai dari awal atau Shift+F5 dari slide aktif. Presenter View otomatis muncul di layar laptop, sementara audiens melihat slide biasa di proyektor.

  • 4

    Manfaatkan fitur tambahan saat presentasi

    Klik ikon pena di Presenter View untuk menggambar/menyorot langsung di slide, klik ikon kaca pembesar untuk zoom ke bagian tertentu, atau klik grid untuk melihat semua slide sekaligus.

💻 Hanya Punya Satu Layar?

Jika presentasi dilakukan dengan hanya satu layar (tanpa proyektor terhubung), tekan Alt+F5 untuk membuka Presenter View secara terpisah sebagai simulasi — berguna untuk latihan presentasi sendiri sebelum hari-H.

Materi 2

Hyperlink Antar Slide: Navigasi Non-Linear

Hyperlink memungkinkan presentasi tidak harus berjalan linear dari slide 1 ke slide terakhir — presenter bisa melompat ke bagian tertentu sesuai respons audiens, atau membuat menu navigasi interaktif.

Slide Menu Utama
Topik A →
Topik B →
Topik C →

Slide menu berisi tombol yang masing-masing terhubung ke bagian presentasi berbeda — audiens/presenter memilih topik sesuai kebutuhan

📑

Link ke Slide Tertentu

Menghubungkan tombol/teks ke slide spesifik dalam presentasi yang sama — dasar dari menu navigasi.

🌐

Link ke Website

Membuka halaman web eksternal saat diklik selama Slide Show — browser akan terbuka otomatis.

📧

Link ke Email

Membuka aplikasi email dengan alamat tujuan sudah terisi — berguna untuk slide kontak/penutup.

📁

Link ke File Lain

Membuka dokumen lain (Word, Excel, PDF) yang tersimpan di komputer yang sama saat presentasi.

Cara Membuat Hyperlink

  • 1

    Pilih teks atau objek

    Pilih teks, shape, atau gambar yang akan menjadi "tombol" link. Klik kanan → Link (atau klik Insert → Link, shortcut Ctrl+K).

  • 2

    Pilih tujuan link

    Di kotak dialog Insert Link, pilih Place in This Document untuk link ke slide lain dalam presentasi yang sama, atau Existing File or Web Page untuk link eksternal.

  • 3

    Pilih slide tujuan spesifik

    Jika memilih "Place in This Document", akan muncul daftar semua slide dalam presentasi — klik slide tujuan, lalu klik OK.

  • 4

    Tambahkan tombol "Kembali"

    Di setiap slide topik, tambahkan shape kecil bertuliskan "Kembali ke Menu" yang di-link balik ke slide menu utama — atau gunakan ikon Action Buttons (Insert → Shapes → Action Buttons) yang sudah tersedia bentuk panah/rumahnya.

Materi 3

Menyisipkan Video ke dalam Slide

💾

Video dari File Lokal

Sisipkan video MP4 dari komputer. Video akan ter-embed langsung ke dalam file PPTX (memperbesar ukuran file).

Insert → Video → This Device
🌐

Video Online (YouTube)

Tempel URL video YouTube — video akan diputar streaming, tidak memperbesar ukuran file PPTX. Perlu koneksi internet saat presentasi.

Insert → Video → Online Video
✂️

Trim Video

Potong bagian awal/akhir video tanpa perlu software edit video terpisah — langsung di dalam PowerPoint.

Playback → Trim Video
▶️

Pengaturan Playback

Atur video agar diputar otomatis (Automatically) atau saat diklik (On Click), serta opsi Loop dan Rewind after Playing.

Playback → Start
⚠️ Pertimbangan Ukuran File

Video yang di-embed dari file lokal akan membesarkan ukuran file PPTX secara signifikan — video 5 menit beresolusi HD bisa menambah puluhan hingga ratusan MB. Jika file akan dikirim via email atau LMS dengan batas ukuran, pertimbangkan menggunakan link YouTube (unlisted) sebagai alternatif.

Materi 4

Mengekspor Presentasi Final

Format Kelebihan Kekurangan Cocok untuk
PDF Ukuran kecil, tampilan terjaga di semua perangkat Animasi & video hilang, jadi statis Dikumpulkan ke dosen
Video (MP4) Animasi & narasi suara dipertahankan Ukuran file besar, proses render lama Presentasi asinkron/rekaman
PowerPoint Show (.ppsx) Langsung membuka mode Slide Show tanpa masuk mode edit Tetap butuh PowerPoint terinstall Dikirim untuk presentasi langsung
Gambar (PNG/JPG per slide) Bisa diunggah satu per satu ke media sosial Kehilangan interaktivitas sepenuhnya Posting media sosial

Cara Merekam Presentasi Menjadi Video

  • 1

    Buka fitur Record Slide Show

    Klik tab Slide Show → Record Slide Show → pilih Record from Beginning atau Record from Current Slide.

  • 2

    Rekam narasi dan gerak kursor

    PowerPoint merekam suara mikrofon, video webcam (opsional), timing perpindahan slide, dan coretan pena secara bersamaan. Klik Record untuk memulai per slide.

  • 3

    Ekspor ke video

    Setelah rekaman selesai, klik File → Export → Create a Video. Pilih kualitas (Full HD, HD, atau Standard) dan durasi tiap slide jika tidak ada rekaman narasi. Klik Create Video.

Materi 5

Checklist Sebelum Presentasi

Sebelum tampil di depan kelas atau mengumpulkan tugas akhir, pastikan semua poin berikut sudah diperiksa:

✓

Semua slide menggunakan tema dan font yang konsisten (via Slide Master)

✓

Tidak ada typo atau kesalahan ejaan — gunakan Spelling Check (F7)

✓

Semua gambar memiliki resolusi yang cukup (tidak pecah/blur)

✓

Animasi dan transisi sudah diuji coba lewat Slide Show penuh

✓

Hyperlink (jika ada) sudah diuji — semua tombol mengarah ke slide yang benar

✓

Video/audio (jika ada) diuji putar dengan volume yang sesuai

✓

Catatan pembicara sudah lengkap untuk Presenter View

✓

Ukuran file sudah dikompres jika terlalu besar (Compress Pictures/Media)

✓

File sudah di-backup dalam 2 format: PPTX (untuk edit) dan PDF (untuk cadangan)

Latihan & Tugas

Praktikum Sesi 12

🛠 Latihan Kelas (40 menit terakhir)

Sempurnakan presentasi dari Sesi 11 menjadi presentasi final yang siap tampil:

  • Presenter View (10 menit): Lengkapi catatan pembicara di setiap slide, lalu uji coba Presenter View melalui Slide Show. Latihan menyampaikan 2–3 slide menggunakan catatan tersebut.
  • Hyperlink Menu (15 menit): Buat 1 slide "Menu Utama" di awal presentasi dengan 3 tombol yang masing-masing terhubung ke bagian berbeda dari presentasi. Tambahkan tombol "Kembali ke Menu" di setiap bagian.
  • Embed & Ekspor (15 menit): Sisipkan 1 video pendek (bisa dari YouTube atau file lokal) ke salah satu slide. Setelah itu, ekspor presentasi lengkap ke format PDF untuk dijadikan cadangan.

📌 Tugas Akhir Blok PowerPoint (dikumpulkan sebelum Sesi 13)

  • Sempurnakan presentasi menjadi presentasi final minimal 12 slide yang mengintegrasikan seluruh materi Blok PowerPoint: layout bervariasi, Slide Master konsisten, SmartArt/tabel/chart, animasi terarah, hyperlink menu, dan minimal 1 video.
  • Lengkapi catatan pembicara di setiap slide untuk keperluan Presenter View.
  • Jalani checklist final yang sudah dipelajari — pastikan semua poin terpenuhi.
  • Ekspor dalam tiga format: PPTX (asli), PDF (cadangan statis), dan video MP4 durasi pendek (rekam minimal 2 menit presentasi menggunakan Record Slide Show).
  • Kumpulkan ketiga file dalam folder NIM_Nama_Sesi12_Final → upload ke LMS/Google Classroom.
