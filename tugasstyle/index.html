🗒 Skenario

Konteks Tugas

Kamu baru saja menyelesaikan Praktik Kerja Lapangan (PKL) selama satu bulan di PT. Solusi Digital Nusantara — sebuah perusahaan teknologi informasi di Surabaya. Pembimbing lapangan memintamu membuat laporan singkat PKL yang terlihat profesional sebelum dikumpulkan ke kampus.

Tugasmu adalah membuat dokumen tersebut dua kali — pertama di Microsoft Word, lalu dengan tampilan yang sama di Google Docs — menggunakan semua teknik Style & Formatting yang sudah dipelajari.

📑 Struktur Dokumen

Dokumen harus memuat minimal 5 bab berikut dengan struktur heading yang benar. Setiap bab wajib memenuhi persyaratan yang tercantum.

1
BAB I — Pendahuluan
Heading 1
Judul bab menggunakan Heading 1 (bukan teks bold biasa). Sub-bagian seperti "1.1 Latar Belakang" menggunakan Heading 2.
Style
Paragraf isi ditulis dengan style Normal/Body, spasi 1,5, alignment Justify.
Format
Berisi minimal 3 sub-bagian: 1.1 Latar Belakang, 1.2 Tujuan PKL, 1.3 Manfaat PKL.
Konten
Di Google Docs: gunakan View → Show document outline untuk memverifikasi struktur heading sudah benar.
Docs
2
BAB II — Profil Perusahaan
Heading 1
Buat logo perusahaan sederhana menggunakan Shape (di Word: Insert → Shapes; di Google Docs: Insert → Drawing → New). Logo harus mengandung minimal 2 shape yang digabungkan.
Shape
Sisipkan tabel profil perusahaan (minimal 5 baris × 2 kolom) berisi: nama, bidang usaha, alamat, telp, website. Beri warna header tabel dengan warna tema dokumen.
Tabel
Nama perusahaan di kop/judul bab menggunakan WordArt (Word) atau efek teks tebal berwarna via Drawing → Word Art (Google Docs).
WordArt
3
BAB III — Kegiatan PKL
Heading 1
Gunakan Numbered List bertingkat (multilevel) untuk merinci kegiatan mingguan. Contoh: 1. Minggu 1 → 1.1. Orientasi → 1.2. Pengenalan sistem.
Word
Buat SmartArt alur kerja (Word: Insert → SmartArt → Process) atau diagram manual (Google Docs: Insert → Drawing → gabungkan Shape + Arrow) yang menggambarkan alur kerja di perusahaan, minimal 4 tahap.
SmartArt
Sisipkan Text Box sebagai "catatan penting" atau highlight di sisi halaman. Beri warna background Text Box yang kontras dengan teks isi dokumen.
Text Box
Minimal 1 paragraf menggunakan first line indent 1,25 cm.
Format
4
BAB IV — Hasil & Analisis
Heading 1
Sisipkan Bar Chart atau Pie Chart yang menggambarkan data relevan (contoh: persentase waktu per jenis kegiatan, jumlah tugas per minggu). Edit data chart secara langsung (Word: via Excel mini; Google Docs: via Google Sheets).
Chart
Gunakan Format Painter / Paint Format untuk menyamakan format minimal 3 sub-heading Heading 2 secara konsisten.
Efficiency
Teks analisis menggunakan Bold untuk kata kunci penting, Italic untuk istilah teknis asing, dan minimal sekali menggunakan teks berformat Highlight (warna kuning atau warna lain) untuk temuan utama.
Format
5
BAB V — Penutup
Heading 1
Berisi 5.1 Kesimpulan dan 5.2 Saran menggunakan Heading 2 yang sudah dimodifikasi tampilannya (warna atau font berbeda dari default).
Style
Tambahkan tanda tangan digital menggunakan Shape (kotak + garis) dan Text Box untuk nama dan NIM di bawah saran.
Shape

📤 Pengumpulan

Kumpulkan dua file dengan nama sesuai format berikut:

⬛ Microsoft Word
NIM_NamaLengkap_Word.docx
Contoh: 20230001_AndiSaputra_Word.docx
🔵 Google Docs
NIM_NamaLengkap_GDocs.pdf
Export dari Google Docs: File → Download → PDF Document
ℹ️
Kedua file berisi konten yang sama. Yang dinilai adalah bagaimana masing-masing aplikasi digunakan untuk menghasilkan tampilan serupa — termasuk perbedaan cara SmartArt dibuat di Google Docs.
✅
Simpan file Word dalam format .docx (bukan .doc). Pastikan semua objek (gambar, chart, shape) sudah tertanam (embedded) dalam file sebelum dikumpulkan.
