🔗 Berbagi Dokumen

Mengatur siapa yang boleh mengakses dan apa yang boleh mereka lakukan dengan dokumen.

👥

Tiga Tingkat Izin Akses

▼
👁 Viewer
Hanya bisa membaca. Tidak bisa mengedit atau berkomentar. Cocok untuk: dokumen final yang disebarkan.
💬 Commenter
Bisa membaca & berkomentar. Tidak bisa mengubah isi teks langsung. Cocok untuk: dosen pembimbing yang mereview.
✏️ Editor
Bisa membaca, berkomentar, dan mengedit. Akses penuh ke dokumen. Cocok untuk: anggota tim yang mengerjakan bersama.
⚠️ Berikan akses Editor hanya kepada orang yang benar-benar perlu mengedit. Sekali seseorang menghapus konten penting, perlu diambil dari riwayat versi.
🔗

Cara Membagikan Dokumen

▼
⬛ Microsoft Word

Word berbagi dokumen via OneDrive (file harus disimpan di cloud terlebih dahulu):

  1. 1Simpan dokumen ke OneDrive: klik File → Save a Copy → OneDrive.
  2. 2Klik tombol Share di pojok kanan atas Ribbon (ikon orang + tanda +).
  3. 3Masukkan alamat email orang yang ingin diundang, pilih level akses (Can edit / Can view), tambahkan pesan jika perlu → klik Send.
  4. 4Atau klik Copy Link untuk membuat tautan yang bisa dibagikan di chat/email.
File → Share → Share with People
⚠️ Dokumen Word yang belum disimpan di OneDrive tidak bisa dibagikan secara langsung. File harus online terlebih dahulu.
🔵 Google Docs

Google Docs selalu tersimpan di cloud secara otomatis:

  1. 1Klik tombol Share (biru, pojok kanan atas).
  2. 2Ketik nama atau alamat email penerima di kotak pencarian.
  3. 3Pilih peran: Viewer, Commenter, atau Editor.
  4. 4Opsional: tambahkan pesan, lalu klik Send.
  5. 5Untuk berbagi via link: klik "Change to anyone with the link" → salin tautan.
💡 Di Google Docs, penerima bisa langsung membuka dokumen tanpa perlu memiliki akun Google — jika link disetel ke "Anyone with the link can view".
🔒

Mengubah Pengaturan Link & Batas Waktu Akses

▼
⬛ Microsoft Word
  1. 1Klik Share → Copy Link → klik ikon pensil di sebelah link.
  2. 2Ubah akses link: Anyone, People in your organization, atau Specific people.
  3. 3Bisa mengatur tanggal kedaluwarsa link (fitur Microsoft 365).
🔵 Google Docs
  1. 1Klik Share → General access.
  2. 2Pilih: Restricted (hanya orang yang diundang) atau Anyone with the link.
  3. 3Pilih level akses link: Viewer, Commenter, atau Editor.
  4. 4Klik Copy link lalu Done.

💬 Menambahkan & Mengelola Komentar

Memberikan catatan, pertanyaan, atau saran pada bagian tertentu tanpa mengubah isi teks langsung.

👁

Tampilan Komentar di Dokumen

▼
CONTOH KOMENTAR

Kegiatan PKL dilaksanakan selama satu bulan di PT. Solusi Digital Nusantara, di Surabaya.

💬
Dosen Pembimbing: Tambahkan alamat lengkap perusahaan, bukan hanya nama kotanya saja.
➕

Menambahkan Komentar

▼
⬛ Microsoft Word
  1. 1Seleksi teks yang ingin diberi komentar (atau cukup klik di posisi yang diinginkan).
  2. 2Cara 1: Tab Review → klik New Comment.
  3. 3Cara 2 (Shortcut): Ctrl+Alt+M.
  4. 4Ketik isi komentar di panel yang muncul di kanan dokumen → tekan Ctrl+Enter atau klik ikon kirim.
Review → New Comment
🔵 Google Docs
  1. 1Seleksi teks yang ingin diberi komentar.
  2. 2Cara 1: Klik ikon 💬 (gelembung komentar) yang muncul di sisi kanan halaman.
  3. 3Cara 2: Menu Insert → Comment.
  4. 4Cara 3 (Shortcut): Ctrl+Alt+M — sama persis dengan Word!
  5. 5Ketik komentar → klik Comment untuk menyimpan.
💡 Gunakan @mention dalam komentar (ketik @ diikuti nama) untuk mengirim notifikasi langsung ke orang tertentu — berlaku di Word (OneDrive) maupun Google Docs.
✅

Membalas & Menyelesaikan Komentar

▼
⬛ Microsoft Word
  1. 1Klik komentar yang ingin dibalas → klik Reply → ketik balasan.
  2. 2Untuk menandai selesai: klik ikon ✓ (Resolve) di pojok kanan atas komentar.
  3. 3Komentar yang resolved disembunyikan tapi tidak dihapus. Bisa dilihat kembali via Review → Show Comments → Show Resolved.
  4. 4Untuk menghapus permanen: klik komentar → Delete.
🔵 Google Docs
  1. 1Klik komentar → klik Reply → ketik balasan → tekan Enter.
  2. 2Untuk menyelesaikan: klik tombol Resolve (muncul saat hover komentar).
  3. 3Komentar resolved disembunyikan. Lihat kembali: klik ikon 💬 (Open comment history) di pojok kanan atas → filter Resolved.
  4. 4Untuk menghapus: klik ⋮ → Delete comment.

Merekam setiap penambahan, penghapusan, dan pemformatan yang dilakukan sehingga bisa ditinjau dan diterima atau ditolak.

👁

Mengaktifkan Track Changes

▼
⬛ Microsoft Word
  1. 1Klik tab Review di Ribbon.
  2. 2Klik Track Changes → tombol akan tampak aktif/tertekan (warna berubah).
  3. 3Shortcut cepat: Ctrl+Shift+E.
  4. 4Setiap perubahan yang dilakukan sekarang akan direkam dan ditandai.
  5. 5Untuk mematikan: klik lagi Track Changes (toggle off).
Review → Track Changes
💡 Untuk mencegah orang lain mematikan Track Changes, lindungi dokumen via Review → Restrict Editing → Allow only this type of editing: Tracked changes. Bisa dikunci dengan password.
🔵 Google Docs

Di Google Docs, fitur ini disebut Suggesting Mode (mode saran), bukan Track Changes:

  1. 1Klik ikon pensil (✏️) di pojok kanan atas dokumen — ini menampilkan mode aktif.
  2. 2Pilih Suggesting (ikon pensil + tanda centang).
  3. 3Semua perubahan kini menjadi saran berwarna yang bisa diterima atau ditolak.
💡 Shortcut ke Suggesting Mode di Google Docs: Ctrl+Alt+Shift+S. Pengguna dengan akses Commenter otomatis masuk ke mode Suggesting.
🖥

Mode Tampilan Track Changes

▼
⬛ Microsoft Word

Tab Review → dropdown Display for Review:

  • •All Markup — tampilkan semua perubahan (default, paling lengkap).
  • •Simple Markup — tanda vertikal di margin kiri, klik untuk buka detail.
  • •No Markup — tampilkan dokumen seolah semua perubahan diterima (pratinjau hasil akhir).
  • •Original — tampilkan dokumen asli sebelum ada perubahan.
🔵 Google Docs

Google Docs menampilkan semua saran secara default. Untuk melihat tampilan akhir:

  • •Klik ikon pensil → pilih Viewing — tampilan bersih tanpa markup saran.
  • •Kembali ke Suggesting untuk melanjutkan memberikan saran.
  • •Mode Editing untuk mengedit langsung tanpa merekam sebagai saran.

✅ Menerima & Menolak Revisi

Meninjau setiap perubahan yang direkam dan memutuskan mana yang diterima atau ditolak.

✅

Terima / Tolak Perubahan Satu per Satu

▼
⬛ Microsoft Word
  1. 1Klik di dalam teks yang berubah (dicoret/berwarna) untuk memilihnya.
  2. 2Tab Review → klik ✓ Accept untuk menerima perubahan tersebut.
  3. 3Atau klik ✗ Reject untuk menolak dan mengembalikan ke teks asal.
  4. 4Gunakan tombol Previous / Next untuk berpindah antar perubahan.
Review → Accept / Reject
💡 Klik kanan langsung pada teks berubah → pilih Accept atau Reject — cara tercepat tanpa perlu ke tab Review.
🔵 Google Docs
  1. 1Klik pada saran (perubahan berwarna) yang ingin ditinjau.
  2. 2Sebuah popup kecil muncul di tepi kanan dengan tombol ✓ dan ✗.
  3. 3Klik ✓ untuk menerima saran atau ✗ untuk menolak.
  4. 4Atau klik panel komentar saran di kanan → Accept / Reject.
⚡

Terima / Tolak Semua Perubahan Sekaligus

▼
⬛ Microsoft Word
  1. 1Tab Review → klik panah kecil ▼ di bawah tombol Accept.
  2. 2Pilih "Accept All Changes" untuk menerima semua — dokumen menjadi bersih tanpa markup.
  3. 3Atau klik ▼ di bawah Reject → "Reject All Changes" untuk mengembalikan ke versi asli sebelum ada revisi.
⚠️ Accept All tidak bisa di-Undo jika dilakukan sembarangan. Simpan salinan dokumen sebelum menerima semua perubahan sekaligus.
🔵 Google Docs
  1. 1Klik menu Tools → Review suggested edits.
  2. 2Panel di atas dokumen muncul dengan tombol Accept all atau Reject all.
  3. 3Bisa juga menelusuri satu per satu dengan tombol Previous / Next di panel yang sama.
Tools → Review suggested edits
🕐

Riwayat Versi — Kembali ke Versi Sebelumnya

▼
⬛ Microsoft Word
  1. 1Klik File → Info → Version History.
  2. 2Panel riwayat muncul di kanan — klik versi yang ingin dilihat.
  3. 3Untuk memulihkan: klik Restore pada versi yang dipilih.
⚠️ Riwayat versi Word hanya tersedia jika dokumen disimpan di OneDrive. File lokal (.docx di komputer) tidak memiliki fitur ini.
🔵 Google Docs
  1. 1Klik File → Version history → See version history.
  2. 2Panel kanan menampilkan semua versi dengan timestamp dan pengguna yang membuat perubahan.
  3. 3Klik versi untuk melihat tampilannya. Klik Restore this version untuk memulihkan.
  4. 4Bisa memberi nama versi penting: klik ⋮ → Name this version.
✅ Riwayat versi Google Docs selalu tersedia secara otomatis, tanpa perlu pengaturan khusus. Ini keunggulan besar Docs dibanding Word.

⚡ Kolaborasi Real-time

Mengedit dokumen bersama secara bersamaan — melihat kursor dan perubahan orang lain langsung.

🖥

Kolaborasi Bersamaan (Co-authoring)

▼
⬛ Microsoft Word
  1. 1Dokumen harus disimpan di OneDrive dan sudah dibagikan kepada editor lain.
  2. 2Saat pengguna lain membuka dokumen yang sama, kursor mereka muncul dengan nama dan warna berbeda.
  3. 3Perubahan antar pengguna tersinkronisasi secara otomatis setiap beberapa detik.
  4. 4Jika ada konflik edit (dua orang mengedit paragraf yang sama), Word akan memunculkan dialog untuk menyelesaikan konflik.
⚠️ Co-authoring Word lebih stabil di browser (Word Online) dibanding Word Desktop untuk sinkronisasi real-time. Gunakan Word Online jika sering ada konflik.
🔵 Google Docs
  1. 1Setelah dokumen dibagikan, semua editor yang membukanya langsung bisa mengedit bersamaan.
  2. 2Kursor pengguna lain terlihat secara real-time dengan nama dan warna masing-masing.
  3. 3Perubahan langsung terlihat tanpa perlu refresh atau simpan manual.
  4. 4Chat langsung tersedia: klik ikon 💬 chat di pojok kanan atas untuk diskusi tanpa meninggalkan dokumen.
✅ Real-time collaboration Google Docs secara konsisten lebih mulus dan cepat dibanding Word — ini kekuatan utama Google Docs untuk kerja tim.
📥

Mode Offline & Sinkronisasi

▼
⬛ Microsoft Word
  1. 1Word Desktop bisa bekerja offline — perubahan disimpan lokal dan disinkronkan ke OneDrive saat kembali online.
  2. 2Indikator sinkronisasi muncul di status bar bawah Word.
  3. 3Konflik pengeditan offline akan dimunculkan saat kembali online.
🔵 Google Docs
  1. 1Aktifkan mode offline: File → Make available offline (membutuhkan ekstensi Chrome Offline).
  2. 2Perubahan offline tersimpan lokal dan disinkronkan otomatis saat online kembali.
  3. 3Indikator "Offline" muncul di atas dokumen saat tidak terhubung internet.
⚖️

💡 Tips Kolaborasi Efektif

🔒

Aktifkan TC Sebelum Berbagi

Jika ingin semua revisi tercatat, aktifkan Track Changes sebelum membagikan dokumen ke orang lain — bukan sesudah mereka mengedit.

💬

@Mention agar Tidak Terlewat

Gunakan @nama saat membuat komentar agar orang yang relevan mendapat notifikasi. Komentar tanpa mention sering tidak terbaca.

🕐

Beri Nama Versi Penting

Di Google Docs maupun Word Online, beri nama pada versi yang sudah disetujui (misal "Versi Draf 2 — Disetujui Dosen") agar mudah ditemukan jika perlu rollback.

👁

Preview Sebelum Accept All

Gunakan tampilan "No Markup" (Word) atau "Viewing" (Docs) untuk melihat bagaimana dokumen akhir akan terlihat sebelum menerima semua perubahan.

✅

Resolve Komentar yang Sudah Ditanggapi

Biasakan klik Resolve setiap komentar yang sudah ditindaklanjuti agar panel komentar tidak penuh dengan diskusi lama yang membingungkan.

📧

Commenter vs Editor

Berikan akses Commenter (bukan Editor) ke dosen atau reviewer — mereka bisa memberikan masukan tanpa bisa menghapus atau mengubah teks langsung.
