🖼 Menyisipkan & Mengatur Gambar

Cara menambahkan gambar dari file, mengatur ukuran, posisi, dan text wrapping.

➕

Menyisipkan Gambar

▼
⬛ Microsoft Word
  1. 1Klik tab Insert di Ribbon.
  2. 2Klik Pictures → pilih This Device (dari komputer) atau Online Pictures.
  3. 3Cari file gambar, klik Insert.
  4. 4Gambar langsung muncul di posisi kursor.
Insert → Pictures → This Device
🔵 Google Docs
  1. 1Klik menu Insert di menu bar atas.
  2. 2Pilih Image → Upload from computer.
  3. 3Pilih file gambar, klik Open.
  4. 4Gambar muncul di dokumen.
Insert → Image → Upload from computer
↔️

Mengubah Ukuran Gambar

▼
⬛ Microsoft Word
  1. 1Klik gambar untuk memilihnya — muncul handle (titik di sudut & sisi).
  2. 2Drag handle sudut untuk memperbesar/memperkecil secara proporsional.
  3. 3Untuk ukuran tepat: tab Picture Format → isi nilai Height & Width.
  4. 4Pastikan kunci proporsi (Lock Aspect Ratio) aktif agar tidak miring.
💡Tahan Shift saat drag handle sudut untuk menjaga proporsi secara manual.
🔵 Google Docs
  1. 1Klik gambar → muncul handle biru di sudut dan sisi.
  2. 2Drag handle sudut biru untuk resize proporsional.
  3. 3Untuk ukuran pasti: klik gambar → Image options → bagian Size & Rotation.
💡Google Docs otomatis menjaga proporsi saat drag dari sudut.
🔄

Text Wrapping (Aliran Teks di Sekitar Gambar)

▼
⬛ Microsoft Word
  1. 1Klik gambar → tab Picture Format.
  2. 2Klik Wrap Text, pilih salah satu:
In Line with Text — gambar seperti karakter teks biasa.
Square — teks mengalir mengikuti kotak gambar.
Tight — teks mengikuti kontur gambar.
Behind Text — gambar di belakang teks.
In Front of Text — gambar menutupi teks.
🔵 Google Docs
  1. 1Klik gambar → muncul toolbar kecil di bawah gambar.
  2. 2Pilih salah satu ikon wrapping:
In line — gambar mengikuti alur teks.
Wrap text — teks mengalir di sekitar gambar.
Break text — gambar memisah paragraf.
⚠️Google Docs tidak mendukung Tight wrapping seperti Word.

🔷 Shape & SmartArt

Membuat diagram visual, bentuk geometris, dan infografis sederhana.

🔷

Menyisipkan Shape

▼
⬛ Microsoft Word
  1. 1Tab Insert → klik Shapes.
  2. 2Pilih bentuk yang diinginkan (kotak, lingkaran, panah, dll.).
  3. 3Drag di dokumen untuk menggambar shape.
  4. 4Gunakan tab Shape Format untuk mengubah warna, border, efek.
💡Klik kanan shape → Add Text untuk menambahkan tulisan di dalam shape.
🔵 Google Docs
  1. 1Klik Insert → Drawing → New.
  2. 2Di jendela Drawing, klik ikon Shape di toolbar.
  3. 3Pilih kategori shape dan gambar di kanvas.
  4. 4Klik Save and Close untuk memasukkan ke dokumen.
⚠️Google Docs menggunakan Drawing Canvas terpisah, bukan langsung di halaman seperti Word.
🗂

Menyisipkan SmartArt

▼
⬛ Microsoft Word
  1. 1Tab Insert → klik SmartArt.
  2. 2Pilih kategori: List, Process, Cycle, Hierarchy, Relationship, dll.
  3. 3Pilih template, klik OK.
  4. 4Klik tiap kotak teks dan ketik isi diagram.
  5. 5Gunakan tab SmartArt Design untuk mengubah tema warna.
Insert → SmartArt → Pilih kategori → OK
🔵 Google Docs
  1. 1SmartArt tidak tersedia di Google Docs secara native.
  2. 2Gunakan Insert → Drawing → New untuk membuat diagram manual.
  3. 3Gabungkan Shape + Text Box + Arrow di Drawing Canvas.
  4. 4Klik Save and Close.
⚠️Ini perbedaan signifikan antara Word dan Google Docs. Di Docs, diagram harus dibuat manual menggunakan Drawing.

📊 Menyisipkan & Mengedit Chart

Menambahkan grafik (bar, pie, line) dan mengubah datanya langsung di dokumen.

📊

Menyisipkan Chart

▼
⬛ Microsoft Word
  1. 1Tab Insert → klik Chart.
  2. 2Pilih jenis chart (Column, Bar, Pie, Line, dll.).
  3. 3Klik OK — Excel mini akan terbuka di samping.
  4. 4Edit data di spreadsheet mini tersebut.
  5. 5Tutup Excel mini — chart otomatis diperbarui.
Insert → Chart → Pilih tipe → OK
🔵 Google Docs
  1. 1Klik Insert → Chart → pilih tipe (Bar, Column, Line, Pie).
  2. 2Chart akan muncul dengan data contoh.
  3. 3Klik ikon Edit in Sheets (pensil) → Google Sheets terbuka di tab baru.
  4. 4Edit data di Sheets, kembali ke Docs.
  5. 5Klik Update pada chart untuk menyinkronkan perubahan.
💡Chart Google Docs terhubung langsung dengan Google Sheets — perubahan data otomatis bisa di-sync kapan saja.
🎨

Mengubah Tampilan Chart

▼
⬛ Microsoft Word
  1. 1Klik chart → tab Chart Design muncul.
  2. 2Gunakan Chart Styles untuk mengubah tema visual.
  3. 3Klik Add Chart Element untuk menambah/hapus: judul, legenda, data label.
  4. 4Double-click elemen chart untuk format detail (warna, font, border).
🔵 Google Docs
  1. 1Klik chart → klik ikon titik tiga (⋮) di pojok kanan atas chart.
  2. 2Pilih Edit chart — Sheets terbuka dengan panel Chart Editor.
  3. 3Tab Setup: ubah tipe chart dan data range.
  4. 4Tab Customize: ubah warna, font, legenda, gridlines.

🅰 Text Box & WordArt

Membuat kotak teks bebas dan judul bergaya artistik.

🅰

Menyisipkan Text Box

▼
⬛ Microsoft Word
  1. 1Tab Insert → klik Text Box.
  2. 2Pilih template siap pakai, atau Draw Text Box untuk menggambar bebas.
  3. 3Drag di halaman untuk membuat kotak, lalu ketik teks.
  4. 4Gunakan tab Shape Format untuk mengubah warna background dan border.
Insert → Text Box → Draw Text Box
🔵 Google Docs
  1. 1Text Box di Google Docs berada di dalam Drawing Canvas.
  2. 2Klik Insert → Drawing → New.
  3. 3Klik ikon Text Box (T) di toolbar Drawing.
  4. 4Drag untuk membuat kotak, ketik teks, atur format.
  5. 5Klik Save and Close.
⚠️Text Box di Google Docs tidak bisa diletakkan langsung di halaman — harus melalui Drawing Canvas.
✨

Menyisipkan WordArt

▼
⬛ Microsoft Word
  1. 1Tab Insert → klik WordArt.
  2. 2Pilih gaya WordArt dari galeri.
  3. 3Ketik teks di kotak "Your text here".
  4. 4Gunakan Shape Format → Text Effects untuk mengubah efek (Shadow, Reflection, Glow, Transform).
Insert → WordArt → Pilih gaya
🔵 Google Docs
  1. 1WordArt ada di dalam Drawing Canvas.
  2. 2Klik Insert → Drawing → New.
  3. 3Klik menu Actions → Word art di toolbar Drawing.
  4. 4Ketik teks → Enter untuk konfirmasi.
  5. 5Atur warna fill, border, dan font di toolbar.

📋 Menyisipkan & Mengatur Tabel

Membuat tabel, mengatur kolom/baris, dan menerapkan format/warna pada sel.

📋

Menyisipkan Tabel

▼
⬛ Microsoft Word
  1. 1Tab Insert → klik Table.
  2. 2Geser mouse di grid untuk memilih jumlah baris × kolom (maks 10×8 via grid).
  3. 3Atau klik Insert Table → isi jumlah baris & kolom secara manual.
  4. 4Klik OK — tabel muncul di dokumen.
🔵 Google Docs
  1. 1Klik Insert → Table.
  2. 2Geser di grid untuk memilih ukuran tabel (maks 20×20).
  3. 3Klik untuk memasukkan tabel ke dokumen.
🎨

Mengatur Format Tabel (Warna, Border, Shading)

▼
⬛ Microsoft Word
  1. 1Klik di dalam tabel → tab Table Design muncul di Ribbon.
  2. 2Pilih salah satu template di Table Styles untuk format cepat.
  3. 3Untuk warna sel: seleksi sel → Shading → pilih warna.
  4. 4Untuk border: klik Borders → atur jenis, ketebalan, warna border.
🔵 Google Docs
  1. 1Seleksi sel → klik kanan → Table properties.
  2. 2Di panel kanan: atur Cell background color untuk shading.
  3. 3Atur Table border untuk warna dan ketebalan border.
  4. 4Atur Cell padding untuk jarak teks dalam sel.
➕

Menambah & Menghapus Baris/Kolom

▼
⬛ Microsoft Word
  1. 1Klik kanan di baris/kolom yang diinginkan.
  2. 2Pilih Insert → Insert Rows Above/Below atau Insert Columns Left/Right.
  3. 3Untuk hapus: klik kanan → Delete Rows atau Delete Columns.
💡Tekan Tab di sel terakhir tabel untuk otomatis menambah baris baru.
🔵 Google Docs
  1. 1Klik kanan di dalam sel tabel.
  2. 2Pilih Insert row above/below atau Insert column left/right.
  3. 3Untuk hapus: klik kanan → Delete row atau Delete column.

➗ Menyisipkan Persamaan Matematika

Menambahkan rumus dan simbol matematika menggunakan equation editor bawaan.

➗

Menyisipkan Persamaan

▼
⬛ Microsoft Word
  1. 1Tab Insert → klik Equation (atau tekan Alt + =).
  2. 2Kotak equation muncul — ketik rumus menggunakan keyboard.
  3. 3Tab Equation di Ribbon berisi simbol dan struktur: fraksi, akar, integral, sigma, dll.
  4. 4Klik di luar kotak untuk selesai.
Insert → Equation → atau tekan Alt + =
💡Word mendukung LaTeX — ketik \frac{a}{b} lalu tekan Space untuk otomatis dirender menjadi fraksi.
🔵 Google Docs
  1. 1Klik Insert → Equation.
  2. 2Toolbar equation muncul di bagian atas dokumen.
  3. 3Pilih kategori simbol dari toolbar: Greek letters, Miscellaneous, Relations, Math operations, Arrows.
  4. 4Ketik teks dan sisipkan simbol dari toolbar.
  5. 5Klik di luar kotak equation untuk selesai.
Insert → Equation
⚠️Google Docs tidak mendukung LaTeX shortcut seperti Word. Simbol harus dipilih manual dari toolbar.

💡 Tips Penting

🔒

Jaga Proporsi Gambar

Selalu resize dari handle sudut, bukan sisi tengah, agar gambar tidak meregang atau memipih.

📐

Alignment Objek

Gunakan Align di Shape Format (Word) atau tahan Shift klik untuk memilih beberapa objek sekaligus, lalu ratakan bersama.

📋

Group Objek

Pilih beberapa shape → klik kanan → Group agar semua objek bergerak bersama tanpa terpisah.

🔄

Undo Selalu Tersedia

Jangan takut bereksperimen. Gunakan Ctrl+Z (Word) atau Cmd+Z untuk membatalkan langkah terakhir.

☁️

Google Docs Auto-Save

Google Docs menyimpan otomatis setiap perubahan. Tidak perlu tekan Ctrl+S — semua perubahan langsung tersimpan ke cloud.

🖨

Cek Sebelum Print

Selalu gunakan Print Preview sebelum mencetak. Objek dengan text wrapping kadang bergeser saat dicetak jika margin tidak cukup.
