📌 Header & Footer

Menambahkan informasi berulang di bagian atas dan bawah setiap halaman — nama penulis, judul dokumen, tanggal, atau nomor halaman.

👁

Seperti Apa Header & Footer di Dokumen

▼
Laporan PKL — PT. Solusi Digital Nusantara HEADER UKWMS 2024
FOOTER Universitas Katolik Widya Mandala Surabaya Hal. 3

Header dan footer muncul otomatis di setiap halaman tanpa perlu diketik ulang.

➕

Menyisipkan Header & Footer

▼
⬛ Microsoft Word
  1. 1Klik tab Insert di Ribbon.
  2. 2Klik Header → pilih template siap pakai, atau klik Edit Header untuk isian kosong.
  3. 3Area header aktif (ditandai garis biru putus-putus). Ketik teks header.
  4. 4Untuk footer: klik Footer → pilih template, atau tekan Tab dua kali dari header untuk loncat ke footer.
  5. 5Klik Close Header and Footer (atau double-click area isi dokumen) untuk keluar dari mode edit header/footer.
Insert → Header / Footer → Edit Header
💡 Double-click area header/footer yang sudah ada untuk mengeditnya kembali tanpa harus masuk menu Insert.
🔵 Google Docs
  1. 1Klik menu Insert → pilih Headers & footers.
  2. 2Pilih Header atau Footer.
  3. 3Kursor otomatis pindah ke area header/footer. Ketik teks yang diinginkan.
  4. 4Klik di area isi dokumen untuk kembali ke mode pengetikan biasa.
Insert → Headers & footers → Header / Footer
💡 Di Google Docs, klik langsung di area atas/bawah halaman (di luar area teks utama) untuk mengaktifkan header/footer secara langsung.
📄

Header Berbeda di Halaman Pertama

▼
⬛ Microsoft Word
  1. 1Double-click area header untuk masuk mode edit.
  2. 2Di tab Header & Footer, centang opsi Different First Page.
  3. 3Header halaman pertama kini terpisah — bisa dikosongkan atau diisi berbeda (misal: halaman judul tidak perlu header).
  4. 4Centang juga Different Odd & Even Pages jika dokumen dicetak bolak-balik.
📌 Untuk laporan akademik, halaman judul dan daftar isi biasanya tidak memiliki header. Gunakan Different First Page atau Section Break untuk mengatur ini.
🔵 Google Docs
  1. 1Klik di area header dokumen untuk mengaktifkannya.
  2. 2Centang opsi Different first page header & footer yang muncul di bawah area header.
  3. 3Halaman pertama kini memiliki header/footer tersendiri yang bisa berbeda.
⚠️ Google Docs hanya mendukung Different First Page. Opsi header ganjil/genap berbeda tidak tersedia secara native.
📐

Mengatur Jarak Header & Footer dari Tepi Halaman

▼
⬛ Microsoft Word
  1. 1Double-click area header → tab Header & Footer muncul.
  2. 2Di grup Position, ubah nilai Header from Top dan Footer from Bottom (standar: 1,25 cm).
🔵 Google Docs
  1. 1Klik Format → Headers & footers…
  2. 2Atur nilai Header (jarak dari atas) dan Footer (jarak dari bawah) dalam satuan cm.
Format → Headers & footers…

🔢

📌 Laporan akademik Indonesia umumnya: halaman pendahuluan/daftar isi → romawi kecil (i, ii, iii), halaman isi/bab → angka arab (1, 2, 3).
➕

Menyisipkan Nomor Halaman

▼
⬛ Microsoft Word
  1. 1Tab Insert → klik Page Number.
  2. 2Pilih posisi: Top of Page (header), Bottom of Page (footer), Current Position.
  3. 3Pilih alignment: kiri, tengah, atau kanan.
  4. 4Nomor halaman otomatis muncul di semua halaman.
Insert → Page Number → Bottom of Page
💡 Untuk format "Halaman 1 dari 5", gunakan Insert → Quick Parts → Field → NumPages dikombinasi dengan Page Number.
🔵 Google Docs
  1. 1Klik di dalam area header atau footer terlebih dahulu.
  2. 2Klik menu Insert → Page numbers.
  3. 3Pilih salah satu dari 4 opsi tampilan (posisi dan apakah halaman pertama ditampilkan).
  4. 4Untuk "Halaman X dari Y": pilih Page count dari menu Insert, lalu ketik teks di sekitarnya.
Insert → Page numbers
⚠️ Di Google Docs, nomor halaman harus disisipkan dari dalam area header/footer. Tidak bisa disisipkan langsung dari area isi dokumen.
🔄

Memulai Ulang Nomor Halaman (Romawi → Angka Arab)

▼
⬛ Microsoft Word

Kunci: harus ada Section Break antara bagian romawi dan bagian angka arab.

  1. 1Tempatkan kursor di awal halaman BAB I (setelah daftar isi).
  2. 2Tab Layout → Breaks → Next Page (Section Break).
  3. 3Double-click footer di section baru → nonaktifkan "Link to Previous" di tab Header & Footer.
  4. 4Klik Page Number → Format Page Numbers…
  5. 5Pilih format angka arab, set Start at: 1 → OK.
  6. 6Kembali ke section pertama → format nomor halaman ke romawi kecil.
⚠️ Langkah nonaktifkan "Link to Previous" sangat penting — tanpa ini, perubahan footer di section baru akan ikut mengubah footer di section sebelumnya.
🔵 Google Docs
  1. 1Klik menu Insert → Page numbers → More options…
  2. 2Di dialog yang muncul, ubah Start at sesuai kebutuhan.
  3. 3Pilih apakah nomor muncul di halaman pertama atau tidak.
⚠️ Google Docs tidak mendukung dua format nomor berbeda (romawi + angka arab) dalam satu dokumen tanpa plugin tambahan. Ini adalah keterbatasan signifikan dibanding Word.
💡 Solusi di Google Docs: buat dua dokumen terpisah (satu untuk bagian romawi, satu untuk isi), lalu gabungkan saat ekspor ke PDF menggunakan tools seperti Adobe Acrobat atau Smallpdf.

✂️ Page Break & Section Break

Memaksa konten pindah ke halaman baru, atau membagi dokumen menjadi beberapa section dengan pengaturan halaman berbeda.

🔍

Perbedaan Page Break vs Section Break

▼
PAGE BREAK Pindah Halaman Saja
Memindahkan konten ke halaman baru. Pengaturan halaman tetap sama — margin, orientasi, header/footer tidak berubah. Paling sering digunakan.
SECTION BREAK Buat Bagian Baru
Membagi dokumen jadi section terpisah. Tiap section bisa punya pengaturan berbeda: orientasi, margin, nomor halaman, header/footer sendiri.
💡 Gunakan Page Break untuk: memulai bab baru di halaman baru. Gunakan Section Break untuk: mengubah orientasi satu halaman menjadi landscape, atau mengganti format nomor halaman.
📄

Menyisipkan Page Break

▼
⬛ Microsoft Word
  1. 1Tempatkan kursor di awal konten yang ingin dipindah ke halaman baru (misal: sebelum judul BAB II).
  2. 2Cara 1 (Shortcut): Tekan Ctrl+Enter — cara tercepat.
  3. 3Cara 2 (Menu): Tab Insert → Page Break.
  4. 4Cara 3 (via Breaks): Tab Layout → Breaks → Page.
⚠️ Hindari membuat "halaman baru" dengan menekan Enter berkali-kali. Ini menyebabkan konten bergeser saat teks diubah. Selalu gunakan Page Break yang tepat.
🔵 Google Docs
  1. 1Tempatkan kursor sebelum konten yang ingin dipindah.
  2. 2Shortcut: Ctrl+Enter — sama dengan Word.
  3. 3Via menu: Insert → Break → Page break.
Insert → Break → Page break
💡 Shortcut Ctrl+Enter untuk Page Break sama persis di Word dan Google Docs — mudah diingat!
✂️

Menyisipkan Section Break

▼
⬛ Microsoft Word
  1. 1Tempatkan kursor di posisi pemisah section.
  2. 2Tab Layout → Breaks → pilih jenis Section Break:
Next Page — section baru dimulai di halaman baru (paling umum).
Continuous — section baru di halaman yang sama (untuk mengubah jumlah kolom).
Even/Odd Page — section baru di halaman genap/ganjil berikutnya.
Layout → Breaks → Next Page
💡 Untuk melihat di mana section break berada: aktifkan Show Formatting Marks dengan Ctrl+Shift+8 (atau ikon ¶ di tab Home).
🔵 Google Docs
  1. 1Klik Insert → Break.
  2. 2Pilih Section break (next page) atau Section break (continuous).
Insert → Break → Section break (next page)
⚠️ Section Break di Google Docs lebih terbatas — tidak mendukung pengaturan header/footer berbeda per section seperti di Word.
⚡

Column Break — Teks Multi-Kolom

▼
⬛ Microsoft Word
  1. 1Seleksi teks yang ingin dibuat multi-kolom.
  2. 2Tab Layout → Columns → pilih 2 atau 3 kolom.
  3. 3Untuk memaksa pindah kolom: Layout → Breaks → Column.
🔵 Google Docs
  1. 1Seleksi teks.
  2. 2Format → Columns → pilih 1, 2, atau 3 kolom.
  3. 3Untuk pindah kolom: Insert → Break → Column break.

📋 Daftar Isi Otomatis

Membuat, memperbarui, dan mengustomisasi daftar isi yang dihasilkan otomatis dari Heading Style dokumen.

👁

Hasil Daftar Isi Otomatis

Titik-titik pemandu (leader dots) dan nomor halaman dibuat otomatis. Cukup klik Update jika konten berubah.

⚠️

Syarat Utama: Semua Judul Harus Menggunakan Heading Style

▼

Daftar isi otomatis hanya bisa dibuat jika judul bab dan sub-bab sudah menggunakan Heading Style (Heading 1, Heading 2, Heading 3), bukan teks biasa yang diformat tebal secara manual.

✅ BENAR — Terdeteksi di Daftar Isi
Klik teks judul bab → terapkan Heading 1 dari panel Styles. Word/Docs akan mendeteksinya otomatis.
❌ SALAH — Tidak Terdeteksi
Teks diketik lalu diformat Ctrl+B dan ukuran diperbesar manual. Tampak seperti judul, tapi bukan Heading Style.
📋

Menyisipkan Daftar Isi

▼
⬛ Microsoft Word
  1. 1Tempatkan kursor di halaman yang akan menjadi halaman daftar isi (biasanya setelah halaman judul).
  2. 2Tab References → klik Table of Contents.
  3. 3Pilih template otomatis: Automatic Table 1 atau Automatic Table 2.
  4. 4Daftar isi langsung muncul dengan titik-titik pemandu dan nomor halaman.
References → Table of Contents → Automatic Table 1
💡 Pilih Custom Table of Contents… untuk mengatur berapa level heading yang ditampilkan (default: 3 level) dan mengubah karakter pemandu titik.
🔵 Google Docs
  1. 1Tempatkan kursor di posisi daftar isi.
  2. 2Klik menu Insert → Table of contents.
  3. 3Pilih salah satu dari 3 gaya tampilan:
    • With page numbers — klasik dengan nomor halaman di kanan
    • With blue links — tanpa nomor, judul bisa diklik
    • Plain text — tanpa nomor dan tanpa link
  4. 4Daftar isi langsung muncul.
Insert → Table of contents
🔄

Memperbarui Daftar Isi setelah Konten Berubah

▼
⬛ Microsoft Word
  1. 1Klik di dalam area daftar isi → muncul kotak abu-abu.
  2. 2Klik tombol Update Table yang muncul di bagian atas daftar isi.
  3. 3Dialog muncul, pilih:
    • Update page numbers only — hanya perbarui nomor halaman
    • Update entire table — perbarui judul dan nomor halaman
  4. 4Pilih Update entire table jika ada judul bab yang berubah.
⚠️ Selalu perbarui daftar isi sebelum mencetak atau mengumpulkan dokumen. Nomor halaman bisa berubah setelah konten diedit.
🔵 Google Docs
  1. 1Klik di dalam daftar isi yang sudah ada.
  2. 2Klik ikon ↺ Refresh (lingkaran panah) yang muncul di pojok kiri atas daftar isi.
  3. 3Daftar isi langsung diperbarui.
💡 Di Google Docs, daftar isi dengan format "With blue links" bisa diklik langsung untuk navigasi ke bab tersebut — sangat berguna untuk dokumen digital.
🎨

Mengustomisasi Tampilan Daftar Isi

▼
⬛ Microsoft Word
  1. 1Klik daftar isi → References → Table of Contents → Custom Table of Contents…
  2. 2Ubah Show levels: berapa level heading yang ditampilkan (1 = hanya H1, 2 = H1+H2, dst.).
  3. 3Ubah karakter Tab leader: titik (.), garis (—), atau kosong.
  4. 4Klik Modify… untuk mengubah font dan ukuran teks tiap level ToC.
🔵 Google Docs
  1. 1Klik daftar isi → klik ikon ⋮ (tiga titik) yang muncul.
  2. 2Pilih Table of contents options untuk mengubah heading mana yang ditampilkan.
  3. 3Untuk mengubah tampilan visual: seleksi teks dalam daftar isi → format manual (font, ukuran, warna).
⚠️ Kustomisasi daftar isi di Google Docs lebih terbatas — tidak ada opsi Tab leader kustom atau Modify style per level seperti di Word.

💡 Tips Penting

🔢

Romawi Dulu, Arab Kemudian

Urutan kerja di Word: buat Section Break → nonaktifkan "Link to Previous" → baru format nomor per section. Jangan balik urutan atau hasilnya kacau.

⛔

Jangan Enter Berkali-kali

Membuat halaman baru dengan menekan Enter 20× adalah kebiasaan buruk. Gunakan Ctrl+Enter untuk Page Break yang selalu tepat posisinya.

🔄

Update Sebelum Dikumpulkan

Selalu perbarui daftar isi sebelum mencetak atau mengumpulkan. Nomor halaman berubah setiap kali konten ditambah atau dihapus.

👁

Aktifkan Show Formatting Marks

Tekan Ctrl+Shift+8 di Word untuk melihat di mana Page Break dan Section Break berada. Sangat membantu debugging layout dokumen.

🔗

Link to Previous — Kunci Section Break

Saat menggunakan Section Break untuk nomor halaman berbeda, selalu nonaktifkan "Link to Previous" di footer section baru. Ini langkah yang paling sering terlewat.

📋

Daftar Isi = Cermin Heading

Daftar isi otomatis hanya sebaik Heading Style yang kamu terapkan. Jika ada judul yang tidak muncul di daftar isi, pastikan judul itu menggunakan Heading 1/2/3, bukan teks biasa.
