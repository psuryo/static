⬛ Bagian A — Microsoft Word
1
Mengapa Tabel untuk Formulir?
Konsep dasar dan keunggulan pendekatan ini

Tabel di Microsoft Word adalah cara paling andal untuk membuat formulir yang rapi dan terstruktur. Berbeda dengan menggunakan tab atau spasi, tabel memberikan kontrol penuh atas posisi label dan kolom isian.

✅ Keunggulan tabel

  • Layout konsisten di semua perangkat
  • Mudah sejajarkan kolom label & isian
  • Bisa merge sel untuk header
  • Dasar ideal untuk form PDF
  • Mudah diedit dan direplikasi

❌ Jangan gunakan ini

  • Tab manual → berantakan saat dicetak
  • Spasi untuk mensejajarkan → bergeser
  • Text box → sulit diatur ulang
  • Underline biasa → tidak terkontrol
💡
Alur pembelajaran ini Kita akan membuat formulir dari nol di Word menggunakan tabel, kemudian mengekspor ke PDF agar bisa diisi secara digital menggunakan field interaktif (teks, checkbox, dropdown).
2
Membuat dan Mengatur Tabel
Cara menyisipkan tabel dan mengonfigurasi dasarnya

A. Menyisipkan tabel

  1. Klik tab Insert (Sisipkan) di Ribbon
  2. Klik tombol Table → pilih dimensi dengan menyeret kotak, atau klik Insert Table… untuk menentukan jumlah kolom dan baris secara manual
  3. Untuk formulir sederhana, mulai dengan tabel 2 kolom: kolom kiri untuk label, kanan untuk isian
  4. Untuk formulir kompleks, pakai 4–6 kolom dan gabungkan (merge) sesuai kebutuhan
⌨️
Shortcut berguna Tekan Tab di sel terakhir tabel untuk menambah baris baru otomatis. Tekan Alt + Shift + ↑↓ untuk memindah baris ke atas/bawah.

B. Operasi tabel penting

🔀
Merge Cells (Gabung Sel)
Seleksi beberapa sel → klik kanan → Merge Cells. Dipakai untuk header judul atau baris keterangan.
✂️
Split Cells (Bagi Sel)
Klik kanan sel → Split Cells. Berguna membagi satu sel menjadi beberapa kolom isian.
↔️
Atur Lebar Kolom
Seret batas kolom, atau klik kanan → Table Properties → masukkan lebar pasti dalam cm.
↕️
Atur Tinggi Baris
Klik kanan baris → Table Properties → tab Row → centang Specify Height dan masukkan nilai.

C. Posisikan tabel di halaman

  1. Klik tanda + di pojok kiri atas tabel untuk memilih seluruh tabel
  2. Buka Table Properties → tab Table → atur Alignment ke Center
  3. Atur Text wrapping ke None agar tabel mengikuti aliran teks normal

Tanda Tangan

Langkah membangun formulir di atas:

  1. Buat tabel 4 kolom × 10 baris (atau sesuai kebutuhan)
  2. Gabungkan semua sel di baris pertama → ketik judul formulir, beri warna latar belakang biru tua
  3. Baris kedua: gabungkan 4 sel → ketik "A. Data Pribadi" sebagai sub-header bagian
  4. Baris data pribadi: kolom 1 = label, kolom 2–4 digabung = isian panjang
  5. Baris tempat/tgl lahir: kolom 1 label, kolom 2 isian, kolom 3 label, kolom 4 isian (4 kolom terpisah)
  6. Untuk baris tanda tangan: atur tinggi baris minimal 2 cm
  7. Hapus border sisi luar atau beri border seragam sesuai desain
⚠️
Hindari baris kosong di antara tabel Jangan menekan Enter untuk memberi jarak antar bagian. Gunakan fitur Space Before/After di pengaturan paragraf, atau gunakan baris terpisah dengan tinggi kecil sebagai pemisah.

Memformat Tampilan Formulir
Border, warna, tipografi, dan baris isian

Format Border Tabel

  1. Pilih seluruh tabel (klik ikon + di pojok kiri atas)
  2. Tab Table Design → Borders → All Borders untuk border merata
  3. Atau pilih No Border → lalu gambar border manual dengan Draw Table
  4. Atur warna border melalui Pen Color di Table Design

Warna Sel (Shading)

  1. Pilih sel atau baris yang ingin diberi warna
  2. Tab Table Design → Shading → pilih warna
  3. Sel label: gunakan abu-abu muda (Light Gray)
  4. Sel header: gunakan warna primer instansi/perusahaan

Membuat baris isian (garis bawah di dalam sel)

  1. Klik di dalam sel isian yang ingin diberi garis
  2. Pilih teks atau tekan Ctrl+A untuk memilih semua isi sel
  3. Tab Home → klik panah dropdown di samping ikon border → pilih Bottom Border
  4. Ini menciptakan kesan garis isian seperti formulir kertas tradisional
🎨
Tips tipografi formulir Gunakan font Arial atau Calibri ukuran 10–11pt untuk isi tabel. Judul formulir: 14–16pt, Bold. Label sel: 10pt, Bold. Pastikan Cell Margins (Table Properties → Options) minimal 0.1 cm agar teks tidak mepet border.

Checklist sebelum finalisasi

  • Semua label terbaca dengan jelas (ukuran font ≥ 10pt)
  • Header formulir menonjol dengan warna atau ukuran font berbeda
  • Kolom isian cukup lebar untuk diisi tangan
  • Border tabel konsisten dan tidak ada garis ganda yang tidak disengaja
  • Margin halaman cukup (minimal 2 cm) agar tidak terpotong saat dicetak
  • Tabel tidak terpotong di antara halaman (atur di Table Properties → Row → uncheck "Allow row to break across pages")

Mengonversi ke PDF dengan Field Isian
Dari Word ke PDF formulir yang bisa diisi secara digital
📋
Dua jalur konversi
Jalur A (Cepat): Ekspor langsung dari Word → PDF. Field isian PDF ditambahkan menggunakan Adobe Acrobat atau Foxit PDF Editor.
Jalur B (Field otomatis): Tambahkan Content Controls di Word terlebih dahulu → simpan sebagai PDF. Beberapa aplikasi PDF bisa mendeteksi field ini.

Jalur A — Ekspor ke PDF

  1. Klik File → Save As → pilih format PDF (*.pdf)
  2. Klik tombol Options… → pastikan "Document structure tags for accessibility" dicentang
  3. Klik OK dan Save
  4. Buka PDF hasil ekspor di Adobe Acrobat (versi Pro/Standard) atau Foxit PDF Editor
  5. Di Acrobat: menu Tools → Prepare Form → Acrobat akan otomatis mendeteksi kolom kosong dan membuat field
  6. Edit nama field, ubah tipe (teks, checkbox, dropdown) sesuai kebutuhan → Save

Jalur B — Content Controls di Word

  1. Aktifkan tab Developer: File → Options → Customize Ribbon → centang Developer
  2. Klik di sel isian tabel, lalu dari tab Developer pilih jenis kontrol:
  3. Klik Aa Rich Text Content Control atau Plain Text Content Control untuk field teks
  4. Klik ikon checkbox untuk field centang
  5. Klik ikon dropdown untuk field pilihan
  6. Setelah semua field ditambahkan, simpan sebagai PDF seperti Jalur A

Jenis field PDF yang tersedia

Tipe Field Kegunaan Cocok untuk Di Word
Text Field Isian teks bebas Nama, alamat, keterangan Plain Text Control
Checkbox Pilihan ya/tidak Jenis kelamin, persetujuan Checkbox Control
Dropdown Pilihan dari daftar Provinsi, status, kategori Combo Box Control
Date Field Pemilih tanggal Tanggal lahir, jadwal Date Picker Control

✅
Formulir siap digunakan! PDF formulir yang sudah memiliki field isian bisa dibuka di Adobe Reader (gratis), diisi secara digital, disimpan, dan dikirim via email — tanpa perlu mencetak.

Aplikasi alternatif untuk mengedit PDF form

🟠
Adobe Acrobat Pro
Fitur paling lengkap. Prepare Form mendeteksi field otomatis. Berbayar, tersedia versi cloud.
🔵
Foxit PDF Editor
Alternatif ringan dan terjangkau. Mendukung pembuatan form field lengkap.
🟢
LibreOffice Draw
Gratis dan open source. Bisa membuat PDF form dasar dengan field teks dan checkbox.
🌐
PDF Escape / Sejda
Berbasis web, tidak perlu install. Cocok untuk menambahkan field sederhana ke PDF yang sudah ada.
🔵 Bagian B — Google Docs

Membuat Formulir dengan Tabel di Google Docs
Teknik layout formulir menggunakan fitur tabel bawaan Google Docs
🔵
Pendekatan Google Docs Google Docs menggunakan tabel untuk layout formulir cetak, mirip dengan Word. Untuk formulir digital yang bisa diisi online, gunakan Google Forms (dibahas di Materi 7). Kombinasi keduanya memberi fleksibilitas untuk cetak maupun digital.

A. Menyisipkan tabel di Google Docs

  1. Buka dokumen Google Docs → klik menu Insert (Sisipkan) di menu bar atas
  2. Pilih Table → arahkan kursor ke grid untuk memilih jumlah kolom dan baris, lalu klik
  3. Untuk formulir sederhana, mulai dengan tabel 2 kolom (label | isian) atau 4 kolom untuk tata letak yang lebih kompleks
  4. Tabel langsung tersisip di posisi kursor. Tekan Tab untuk berpindah antar sel, dan Tab di sel terakhir untuk menambah baris baru

B. Merge cells (gabung sel) di Google Docs

  1. Klik dan seret untuk memilih beberapa sel yang akan digabungkan
  2. Klik kanan pada sel yang dipilih → pilih Merge cells
  3. Atau gunakan menu Format → Table → Merge cells
  4. Untuk membatalkan: pilih sel yang digabung → klik kanan → Unmerge cells
💡
Cara cepat menambah baris/kolom Arahkan kursor ke tepi kiri (untuk baris) atau tepi atas (untuk kolom) tabel — akan muncul tombol + berwarna biru. Klik untuk menambahkan baris atau kolom di posisi tersebut tanpa perlu klik kanan.

C. Mengatur lebar kolom dan tinggi baris

  1. Seret batas kolom di dalam tabel untuk mengubah lebar secara manual
  2. Untuk lebar presisi: klik kanan sel → Table properties → di bagian Column masukkan lebar dalam cm atau inci
  3. Tinggi baris: klik kanan → Table properties → bagian Row → atur minimum row height
  4. Centang Pin header rows jika tabel panjang dan ingin baris pertama tetap terlihat saat scroll

D. Memformat tampilan tabel formulir

  1. Klik kanan sel → Table properties → tab Color untuk mengatur warna border dan warna latar sel (Cell background color)
  2. Untuk header formulir: pilih baris pertama → beri warna latar biru gelap dan warna teks putih melalui toolbar teks biasa
  3. Atur Cell padding (jarak teks dari tepi sel) minimal 0.1 cm agar teks tidak mepet
  4. Pilih Table border → atur ketebalan (0.5pt untuk tipis, 1–2pt untuk tegas) dan warna border
  5. Untuk menghilangkan border tertentu: pilih sel → Table properties → atur border sisi tertentu ke warna putih atau ketebalan 0

E. Menyimpan sebagai PDF dari Google Docs

  1. Klik menu File → Download → pilih PDF Document (.pdf)
  2. PDF langsung terunduh — tidak ada opsi pengaturan tambahan seperti di Word
  3. Buka PDF hasil unduhan di Adobe Acrobat atau Foxit PDF Editor untuk menambahkan field isian interaktif
  4. Alternatif: gunakan File → Print → pilih Save as PDF (melalui print dialog) untuk kontrol lebih atas orientasi dan ukuran halaman
⚠️
Perbedaan penting vs Microsoft Word Google Docs tidak memiliki fitur Content Controls atau Developer Tab. Field isian interaktif (text field, dropdown) harus ditambahkan di aplikasi PDF editor terpisah setelah ekspor. Untuk formulir digital yang langsung bisa diisi online tanpa PDF, gunakan Google Forms (Materi 7).
7
Membuat Formulir dengan Google Forms
Formulir online interaktif yang langsung bisa diisi dan dikumpulkan datanya
📊
Apa itu Google Forms? Google Forms adalah aplikasi gratis dari Google untuk membuat formulir online. Responden mengisi langsung di browser — tanpa perlu mengunduh file. Semua jawaban terkumpul otomatis di Google Sheets yang bisa dianalisis langsung.

A. Membuat formulir baru

  1. Buka forms.google.com → klik tombol + (Formulir Kosong) atau pilih template yang tersedia
  2. Atau dari Google Drive: klik + Baru → Lainnya → Google Formulir
  3. Isi judul formulir di bagian atas dan deskripsi singkat tujuan formulir
  4. Klik ikon ⚙ (Setelan/Settings) untuk mengatur: siapa yang bisa mengisi, batas pengisian, pengumpulan email responden

B. Menambahkan pertanyaan/field

  1. Klik tombol + di panel kanan untuk menambah pertanyaan baru
  2. Ketik teks pertanyaan (label field) di kotak yang muncul
  3. Klik dropdown jenis jawaban untuk memilih tipe field:
  4. Aktifkan toggle Wajib diisi untuk field yang tidak boleh dikosongkan
  5. Klik ikon ⋮ di setiap pertanyaan untuk opsi tambahan: deskripsi bantuan, validasi input, logika percabangan

C. Mengorganisasi dengan section (bagian)

  1. Klik ikon ≡+ (Tambahkan bagian) di panel kanan untuk membuat halaman baru dalam formulir
  2. Beri judul tiap bagian, misalnya: "Data Pribadi", "Kontak", "Pilihan Program"
  3. Di tiap bagian, atur Go to section based on answer untuk membuat formulir bercabang (conditional logic)
  4. Contoh: jika responden memilih "Mahasiswa" → loncat ke bagian data kampus; jika "Umum" → loncat ke bagian lain

D. Mengatur tampilan (tema) formulir

  1. Klik ikon 🎨 (Sesuaikan tema) di toolbar kanan atas
  2. Pilih warna header, warna latar, dan font yang sesuai identitas institusi
  3. Unggah gambar atau logo untuk dijadikan header formulir
  4. Pratinjau hasil dengan klik ikon 👁 (Pratinjau) — formulir ditampilkan persis seperti yang akan dilihat responden

E. Membagikan dan mengumpulkan respons

  1. Klik tombol Kirim (biru, pojok kanan atas) untuk membagikan formulir
  2. Via email: masukkan alamat email responden langsung dari dialog ini
  3. Via link: salin URL formulir → bagikan melalui WhatsApp, LMS, atau media lain. Klik Persingkat URL untuk link yang lebih pendek
  4. Via embed: salin kode HTML untuk menyematkan formulir di website/portal kampus
  5. Respons masuk bisa dilihat di tab Respons dalam editor formulir, atau klik ikon Sheets untuk membuka semua jawaban di Google Sheets secara real-time
✅
Keunggulan Google Forms untuk data collection Semua respons otomatis tersimpan di Google Sheets. Tersedia ringkasan grafis otomatis (pie chart, bar chart) di tab Respons. Formulir bisa dibagikan tanpa batas responden, dan tidak memerlukan instalasi apapun dari sisi responden.

Pilih Word jika…
Butuh formulir cetak dengan layout presisi, atau formulir PDF interaktif untuk dikirim via email dan diisi dengan Adobe Reader.
🔵
Pilih Google Docs jika…
Tim perlu berkolaborasi membuat formulir bersama secara online, dan hasilnya akan dicetak atau diekspor ke PDF sederhana.
📊
Pilih Google Forms jika…
Perlu mengumpulkan banyak respons secara online, analisis otomatis, tanpa repot mengelola file dari tiap responden.
