💡 Konsep Dasar Mail Merge

Memahami tiga komponen utama mail merge sebelum mulai praktik: template, sumber data, dan hasil gabungan.

🧩

Tiga Komponen Mail Merge

▼
1
📄 Dokumen Utama (Template) Surat, undangan, atau label yang isinya sama untuk semua penerima, kecuali bagian yang berubah (nama, alamat, dll.) — ditandai dengan «Field».
2
📊 Sumber Data (Data Source) Tabel berisi data yang berbeda untuk tiap penerima — biasanya dari Excel, Google Sheets, atau tabel yang dibuat langsung di Word.
3
📑 Hasil Gabungan (Merged Document) Dokumen akhir — satu halaman/surat untuk setiap baris data. Jika data berisi 30 baris, hasilnya 30 surat berbeda.
💡 Bayangkan mail merge seperti mesin fotokopi pintar — bentuk suratnya sama, tapi nama dan detail penerima otomatis berganti setiap kali "dicetak".
📋

Kapan Menggunakan Mail Merge?

▼
  • ✓Mengirim surat undangan ke ratusan tamu dengan nama yang berbeda di tiap surat.
  • ✓Membuat sertifikat peserta pelatihan/seminar untuk puluhan peserta sekaligus.
  • ✓Mencetak label amplop dengan alamat berbeda untuk pengiriman massal.
  • ✓Mengirim surat tagihan atau pemberitahuan dengan nominal yang berbeda per penerima.
  • ✓Membuat kartu nama atau ID card untuk banyak karyawan/mahasiswa dari satu template.
⚖️

Mail Merge di Word vs Google Docs — Gambaran Umum

▼
⬛ Microsoft Word

Memiliki fitur Mail Merge bawaan yang sangat lengkap — wizard langkah demi langkah, mendukung sumber data dari Excel, akses langsung dari menu Mailings. Cocok untuk pemula maupun pengguna lanjutan.

🔵 Google Docs

Tidak punya fitur Mail Merge bawaan. Solusi: menggunakan Google Apps Script sederhana, atau add-on dari Google Workspace Marketplace seperti "Mail Merge for Gmail/Docs".

⚠️ Ini perbedaan paling signifikan di modul ini — Google Docs butuh add-on tambahan, sementara Word sudah siap pakai.

📊 Persiapan Sumber Data

Menyiapkan tabel data yang rapi sebelum memulai proses mail merge — kunci keberhasilan ada di sini.

📐

Aturan Membuat Sumber Data yang Benar

▼

Contoh struktur data yang benar untuk surat undangan:

NamaJabatanPerusahaanKota
Andi SaputraManajer HRDPT Solusi DigitalSurabaya
Budi SantosoKepala Divisi ITCV Maju JayaMalang
Citra DewiDirektur OperasionalPT Sinar AbadiJember
  • ✓Baris pertama = nama kolom (header) — jangan ada baris kosong di atasnya.
  • ✓Satu kolom hanya berisi satu jenis data (jangan campur nama dan kota dalam satu kolom).
  • ✓Nama kolom tidak boleh ada spasi di awal/akhir dan sebaiknya tanpa karakter spesial.
  • ✓Pastikan tidak ada baris atau kolom kosong di tengah data.
  • ✓Simpan data sebagai file terpisah dari dokumen template (Excel/Sheets, bukan di dalam Word).
🗂

Sumber Data yang Bisa Digunakan

▼
⬛ Microsoft Word
  • •Microsoft Excel (.xlsx) — paling umum dan direkomendasikan.
  • •Outlook Contacts — untuk surat ke daftar kontak email.
  • •Tabel di dalam Word — bisa dibuat manual, tapi kurang fleksibel untuk data besar.
  • •File CSV — kompatibel jika diekspor dengan format yang benar.
🔵 Google Docs
  • •Google Sheets — sumber data utama, harus berada di Google Drive yang sama.
  • •Sheet harus memiliki header di baris pertama, sama seperti aturan Excel.
📌 Add-on Mail Merge di Google Docs umumnya membaca data dari Google Sheets — pastikan sheet sudah di-share atau berada di akun yang sama dengan dokumen.
📄

Menyiapkan Dokumen Template

▼

Buat surat seperti biasa, tapi sisakan bagian yang akan diisi otomatis. Contoh template surat undangan:

TEMPLATE

Surabaya, 30 Juni 2026

Kepada Yth.
«Nama»
«Jabatan», «Perusahaan»

💡 Format dokumen template sebelum menyisipkan field merge — atur font, margin, dan heading style terlebih dahulu agar hasil akhirnya konsisten.

🔗 Menjalankan Proses Mail Merge

Langkah lengkap menghubungkan template dengan sumber data hingga menghasilkan dokumen akhir.

⬛

Microsoft Word — Mail Merge Wizard (Lengkap)

▼
1
Mulai Mail Merge Tab Mailings → Start Mail Merge → pilih jenis dokumen: Letters, Email Messages, Envelopes, atau Labels.
2
Hubungkan Sumber Data Klik Select Recipients → Use an Existing List… → cari dan pilih file Excel yang sudah disiapkan → pilih sheet yang berisi data.
3
Edit Daftar Penerima (Opsional) Klik Edit Recipient List untuk menyaring/mengurutkan data, atau menghapus baris tertentu yang tidak ingin disertakan.
4
Sisipkan Merge Field Tempatkan kursor di posisi yang ingin diisi data → klik Insert Merge Field → pilih nama kolom (misal: «Nama»). Ulangi untuk semua field yang dibutuhkan.
5
Pratinjau Hasil Klik Preview Results untuk melihat bagaimana surat akan tampil dengan data asli. Gunakan tombol panah ◀ ▶ untuk melihat data lain.
6
Selesaikan & Gabungkan Klik Finish & Merge → pilih Edit Individual Documents… untuk membuat satu file berisi semua surat, atau Print Documents… untuk langsung mencetak.
Mailings → Start Mail Merge → Letters
⚠️ Pilih "All" saat dialog Merge to New Document muncul untuk menggabungkan semua baris data. Pilih "Current Record" hanya jika ingin satu surat untuk data yang sedang ditampilkan.
🔵

Google Docs — Mail Merge via Add-on

▼
1
Pasang Add-on Mail Merge Buka Google Docs → klik ikon puzzle (Extensions) → Add-ons → Get add-ons → cari "Mail Merge" (contoh: Mail Merge for Google Docs) → klik Install.
2
Siapkan Sheet Data Buka Google Sheets yang berisi data penerima. Pastikan header kolom jelas (Nama, Alamat, dll.) di baris pertama.
3
Tulis Template dengan Placeholder Di Google Docs, ketik teks template. Tandai bagian yang akan diganti dengan format {{Nama}} atau «Nama» sesuai ketentuan add-on yang dipakai.
4
Jalankan Add-on Klik Extensions → Mail Merge → Start. Add-on akan meminta memilih Google Sheets sumber data dan mencocokkan kolom dengan placeholder di dokumen.
5
Pratinjau & Jalankan Merge Cek pratinjau hasil gabungan untuk beberapa baris data. Jika sudah benar, klik Merge atau Send (jika add-on juga mendukung pengiriman email langsung).
6
Unduh / Simpan Hasil Hasil biasanya tersimpan sebagai dokumen baru di Google Drive, satu file per penerima, atau digabung menjadi satu file PDF besar — tergantung add-on yang digunakan.
⚠️ Setiap add-on memiliki format placeholder dan langkah yang sedikit berbeda. Selalu baca dokumentasi add-on yang dipilih sebelum memulai.
🔧

🏷 Mail Merge untuk Label & Amplop

Variasi mail merge untuk mencetak label alamat dan amplop dalam jumlah banyak.

🏷

Membuat Label Alamat (Word)

▼
1
Pilih Jenis Dokumen Label Tab Mailings → Start Mail Merge → Labels…
2
Pilih Ukuran Label Pilih Label vendors (misal Avery US Letter) dan Product number sesuai kertas label fisik yang akan dicetak.
3
Hubungkan Sumber Data & Sisipkan Field Sama seperti surat — Select Recipients lalu Insert Merge Field di label pertama saja.
4
Update Semua Label Klik Update Labels agar field yang sama tersalin ke semua kotak label di halaman.
5
Preview & Finish Preview Results untuk cek tampilan, lalu Finish & Merge seperti biasa.
💡 Jangan lupa klik Update Labels setelah menyisipkan field di kotak pertama — tanpa langkah ini, hanya label pertama yang terisi data, sisanya kosong.
✉️

Membuat Amplop (Word)

▼
  • 1Tab Mailings → Start Mail Merge → Envelopes…
  • 2Pilih ukuran amplop yang sesuai (misal: DL, C5, atau ukuran kustom).
  • 3Hubungkan sumber data, lalu sisipkan field alamat pengirim dan penerima.
  • 4Preview hasil, lalu Finish & Merge.
🔵

Label & Amplop di Google Docs

▼
⚠️ Google Docs tidak memiliki fitur bawaan untuk label maupun amplop. Solusi yang umum digunakan: add-on khusus seperti "Create Labels" atau "Avery Label Merge" yang tersedia di Google Workspace Marketplace, atau membuat layout label secara manual menggunakan tabel di Google Docs lalu menggabungkannya dengan add-on Mail Merge biasa.

💡 Tips Penting

🧹

Rapikan Data Dulu

90% masalah mail merge berasal dari data sumber yang tidak rapi. Cek baris kosong, header ganda, dan konsistensi format sebelum memulai.

👁

Selalu Preview Sebelum Finish

Klik Preview Results dan cek minimal 3-5 baris data berbeda untuk memastikan semua field terisi dengan benar sebelum menggabungkan semua data.

💾

Simpan Template Terpisah

Simpan file template (.docx dengan field merge) terpisah dari hasil akhir (.docx hasil gabungan). Template bisa dipakai ulang untuk data baru di lain waktu.

🔢

Tutup File Excel Saat Merge

Pastikan file Excel sumber data dalam keadaan tertutup saat memilihnya di Word — file yang sedang terbuka kadang tidak terbaca dengan benar.

🏷

Jangan Lupa Update Labels

Saat membuat label, field hanya muncul di kotak pertama. Klik "Update Labels" agar tersalin ke semua kotak — langkah yang paling sering terlewat.

📧

Tes dengan Sedikit Data Dulu

Sebelum menjalankan merge untuk 200 data sekaligus, coba dulu dengan 2-3 baris data untuk memastikan format dan field sudah benar.
