Menerapkan Heading Style

▼
⬛ Microsoft Word
  1. 1Klik pada baris teks yang ingin dijadikan heading, atau seleksi teksnya.
  2. 2Di tab Home, temukan grup Styles di Ribbon.
  3. 3Klik salah satu: Heading 1, Heading 2, Heading 3, atau Normal.
  4. 4Style langsung diterapkan ke teks tersebut.
  5. 5Untuk melihat semua style: klik tombol ▼ di pojok kanan grup Styles.
Home → Styles → Heading 1 / 2 / 3
💡Gunakan shortcut: Ctrl+Alt+1 untuk Heading 1, Ctrl+Alt+2 untuk Heading 2.
🔵 Google Docs
  1. 1Klik pada baris teks yang ingin dijadikan heading.
  2. 2Klik dropdown Normal text di toolbar kiri atas (default menampilkan "Normal text").
  3. 3Pilih: Heading 1, Heading 2, Heading 3, atau Normal text.
  4. 4Style langsung diterapkan.
Dropdown "Normal text" → Pilih Heading
💡Shortcut: Ctrl+Alt+1 (Heading 1), Ctrl+Alt+2 (Heading 2) — sama seperti Word!
🖌

Memodifikasi Style (Mengubah Tampilan Heading)

▼
⬛ Microsoft Word
  1. 1Di grup Styles (tab Home), klik kanan pada style yang ingin diubah (misal: Heading 1).
  2. 2Pilih Modify…
  3. 3Di jendela Modify Style, ubah: font, ukuran, warna, bold/italic, spacing.
  4. 4Centang "Automatically update" agar semua heading yang sama ikut berubah.
  5. 5Klik OK.
✅Mengubah style otomatis memperbarui semua teks dengan style tersebut di seluruh dokumen — jauh lebih efisien dari format manual satu per satu.
🔵 Google Docs
  1. 1Format salah satu teks Heading sesuai keinginan (font, ukuran, warna).
  2. 2Dengan teks itu masih terseleksi, klik dropdown style di toolbar.
  3. 3Arahkan kursor ke style yang ingin diperbarui (misal Heading 1).
  4. 4Klik tombol panah → di sebelah kanannya.
  5. 5Pilih "Update 'Heading 1' to match".
⚠️Di Google Docs, perubahan style hanya berlaku untuk dokumen saat ini, bukan semua dokumen baru. Untuk menyimpan ke default, pilih Options → Save as my default styles.
🗂

Manfaat Heading: Navigation Pane & Outline

▼
⬛ Microsoft Word
  1. 1Tab View → centang Navigation Pane.
  2. 2Panel kiri menampilkan seluruh struktur dokumen berdasarkan Heading.
  3. 3Klik judul di panel untuk langsung loncat ke bagian tersebut.
View → Navigation Pane
🔵 Google Docs
  1. 1Klik View → Show document outline.
  2. 2Panel kiri menampilkan outline berdasarkan Heading yang ada.
  3. 3Klik judul di panel untuk navigasi cepat.
View → Show document outline
💡Heading yang konsisten juga menjadi dasar pembuatan Daftar Isi otomatis (dibahas di Modul 3).

¶ Paragraph Formatting

Mengatur spasi baris, indentasi, alignment, dan list untuk keterbacaan dokumen yang optimal.

↕️

Spasi Baris & Spasi Antar Paragraf

▼
⬛ Microsoft Word
  1. 1Seleksi paragraf atau klik di dalam paragraf.
  2. 2Tab Home → grup Paragraph → klik ikon Line and Paragraph Spacing (ikon baris dengan panah).
  3. 3Pilih nilai: 1.0, 1.5, 2.0, atau klik Line Spacing Options… untuk pengaturan manual.
  4. 4Di Line Spacing Options: atur juga Space Before dan Space After paragraf (dalam pt).
Home → Paragraph → Line Spacing
Spasi 1.0 (Single)
Ini contoh teks dengan spasi baris 1.0. Terasa rapat dan padat, cocok untuk catatan singkat atau tabel.
Spasi 1.5
Ini contoh teks dengan spasi baris 1.5. Lebih nyaman dibaca dan umum digunakan untuk laporan dan skripsi.
Spasi 2.0 (Double)
Ini contoh teks dengan spasi baris 2.0. Memberikan ruang untuk anotasi atau revisi tulisan tangan.
🔵 Google Docs
  1. 1Seleksi paragraf yang ingin diubah.
  2. 2Klik menu Format → Line & paragraph spacing.
  3. 3Pilih nilai spasi: Single, 1.15, 1.5, Double.
  4. 4Untuk spasi kustom: klik Custom spacing… dan isi nilai.
  5. 5Atur juga Space before paragraph dan Space after paragraph.
Format → Line & paragraph spacing
💡Shortcut cepat di Google Docs: Ctrl+Shift+1 (Single), Ctrl+Shift+2 (Double), Ctrl+Shift+5 (1.5).
⬅️

Indentasi & Alignment

▼
⬛ Microsoft Word

Alignment:

  1. 1Seleksi paragraf → tab Home → grup Paragraph.
  2. 2Pilih ikon alignment: ⬅ Kiri, ↔ Tengah, ➡ Kanan, ☰ Justify.
Ctrl+LRata kiri
Ctrl+ETengah
Ctrl+RRata kanan
Ctrl+JJustify

Indentasi:

  1. 1Gunakan tombol Increase/Decrease Indent di grup Paragraph.
  2. 2Atau atur manual lewat Paragraph → Indentation → Left/Right/Special (First line).
🔵 Google Docs

Alignment:

  1. 1Seleksi teks → ikon alignment di toolbar (⬅ ↔ ➡ ☰).
  2. 2Shortcut sama persis dengan Word: Ctrl+L/E/R/J.

Indentasi:

  1. 1Klik ikon Increase/Decrease indent di toolbar.
  2. 2Atau: Format → Align & indent → Indentation options untuk mengatur nilai spesifik dan first line indent.
💡Untuk indentasi awal paragraf (first line), pilih Special indent → First line dengan nilai 1.25 cm (standar akademik Indonesia).
📋

Membuat List (Bullets & Numbering)

▼
⬛ Microsoft Word
  1. 1Tab Home → grup Paragraph → klik ikon Bulleted List atau Numbered List.
  2. 2Untuk memilih gaya bullet/angka: klik panah kecil ▼ di sebelah ikon.
  3. 3Untuk sub-list: tekan Tab di awal baris (naik level) atau Shift+Tab (turun level).
  4. 4List bertingkat (Multilevel): klik Multilevel List → pilih format.
💡Gunakan Multilevel List untuk daftar bertingkat seperti: 1. → 1.1. → 1.1.1. yang sering dipakai di laporan akademik.
🔵 Google Docs
  1. 1Klik ikon Bulleted list atau Numbered list di toolbar.
  2. 2Klik panah kecil ▼ di sebelah ikon untuk memilih gaya yang berbeda.
  3. 3Tekan Tab untuk membuat sub-list, Shift+Tab untuk naik level.
⚠️Google Docs tidak punya fitur Multilevel List bawaan seperti Word. List bertingkat hanya bisa dibuat dengan Tab secara manual.

🔤 Font & Format Karakter

Mengatur jenis font, ukuran, warna, highlight, dan efek teks untuk penekanan yang tepat.

🔤

Mengubah Font, Ukuran & Warna

▼
⬛ Microsoft Word
  1. 1Seleksi teks → tab Home → grup Font.
  2. 2Klik dropdown nama font → ketik nama font atau scroll pilih.
  3. 3Klik dropdown ukuran → ketik angka atau pilih dari daftar.
  4. 4Klik ikon A▼ (Font Color) → pilih warna dari color picker.
  5. 5Klik ikon penanda kuning (Text Highlight Color) untuk highlight.
💡Tekan Ctrl+D untuk membuka dialog Font lengkap dengan semua opsi termasuk efek seperti strikethrough, superscript, subscript, dan small caps.
🔵 Google Docs
  1. 1Seleksi teks → di toolbar atas temukan dropdown nama font.
  2. 2Klik nama font → cari atau scroll pilih font yang diinginkan.
  3. 3Ubah ukuran di kotak angka di sebelah nama font.
  4. 4Klik ikon A▼ untuk warna teks, atau ikon highlight untuk warna latar teks.
💡Klik More fonts di dropdown font untuk mengakses ratusan Google Fonts tambahan yang bisa ditambahkan ke daftar.

Format Painter — Salin Format Cepat

▼
⬛ Microsoft Word
  1. 1Seleksi teks yang formatnya ingin disalin.
  2. 2Tab Home → klik ikon Format Painter (ikon kuas cat 🖌).
  3. 3Kursor berubah menjadi kuas — drag ke teks tujuan.
  4. 4Format disalin seketika.
💡Double-click Format Painter untuk mengunci mode kuas — bisa menerapkan ke banyak bagian sekaligus. Tekan Esc untuk berhenti.
🔵 Google Docs
  1. 1Seleksi teks sumber (yang formatnya ingin disalin).
  2. 2Klik ikon Paint format (ikon roller cat 🎨) di toolbar.
  3. 3Drag ke teks tujuan untuk menerapkan format.
💡Sama seperti Word, double-click ikon Paint format untuk mengunci mode dan menerapkan ke banyak bagian berturut-turut.

🎨 Theme & Warna Dokumen

Menerapkan tema warna dan font yang konsisten ke seluruh dokumen dalam satu klik.

🎨

Menerapkan Document Theme

▼
⬛ Microsoft Word
  1. 1Klik tab Design di Ribbon.
  2. 2Di bagian Document Formatting, scroll dan pilih salah satu tema dari galeri.
  3. 3Pratinjau tema langsung muncul di dokumen saat hover.
  4. 4Klik tema untuk menerapkan.
  5. 5Untuk ubah hanya warna atau font: klik Colors atau Fonts di sebelah kanan galeri tema.
Design → Document Formatting → Pilih tema
✅Tema Word mengubah font Heading, warna aksen, dan style seluruh dokumen sekaligus — perubahan yang konsisten tanpa format manual.
🔵 Google Docs
  1. 1Klik menu Format → Document styles… (atau klik nama style di toolbar → Options).
  2. 2Google Docs tidak memiliki tema bawaan sebanyak Word, namun Anda bisa mengubah font default dan warna heading secara global.
  3. 3Setelah mengubah semua style heading sesuai keinginan, klik dropdown Styles → Options → Save as my default styles.
⚠️Google Docs tidak memiliki tab Design seperti Word. Kustomisasi tema harus dilakukan manual per style, lalu disimpan sebagai default.
💡Add-on pihak ketiga seperti Docs Paragraph Styles+ dapat menambahkan fitur tema yang lebih lengkap ke Google Docs.
🖼

Page Color & Watermark

▼
⬛ Microsoft Word
  1. 1Page Color: Tab Design → Page Color → pilih warna latar halaman.
  2. 2Watermark: Tab Design → Watermark → pilih template (DRAFT, CONFIDENTIAL, dll.) atau buat kustom.
  3. 3Untuk hapus watermark: Watermark → Remove Watermark.
⚠️Warna halaman tidak ikut tercetak jika opsi "Background colors and images" tidak diaktifkan di pengaturan print.
🔵 Google Docs
  1. 1Page Color: File → Page setup → klik kotak Page color → pilih warna.
  2. 2Watermark: Insert → Watermark → ketik teks atau pilih gambar sebagai watermark.
  3. 3Atur transparansi, ukuran, dan sudut rotasi watermark.
💡Fitur Watermark di Google Docs (tersedia sejak 2022) lebih sederhana dari Word, namun sudah cukup untuk kebutuhan umum seperti cap "DRAFT" atau "RAHASIA".

📄 Membuat & Menggunakan Template

Menyimpan format dokumen sebagai template agar bisa digunakan berulang kali tanpa setting ulang.

💾

Menyimpan Dokumen sebagai Template

▼
⬛ Microsoft Word
  1. 1Buat dokumen dengan semua style, margin, dan format yang diinginkan.
  2. 2Klik File → Save As.
  3. 3Di dropdown Save as type, pilih Word Template (*.dotx).
  4. 4Word otomatis mengarahkan ke folder Custom Office Templates. Beri nama file → Save.
  5. 5Untuk menggunakan: File → New → Personal → pilih template yang disimpan.
File → Save As → Word Template (.dotx)
💡Format .dotx (bukan .docx) adalah file template Word. Saat dibuka, Word membuat salinan baru dokumen sehingga template asli tidak pernah tertimpa.
🔵 Google Docs
  1. 1Buat dokumen dengan semua style yang diinginkan.
  2. 2Klik File → Make a copy untuk menduplikasi dokumen sebagai "master template".
  3. 3Simpan di Google Drive dengan nama jelas, misal: "Template Laporan PKL 2024".
  4. 4Setiap kali butuh dokumen baru: buka template → File → Make a copy → isi konten di salinan baru.
💡Untuk template tim/organisasi, gunakan Google Workspace Templates (tersedia di docs.google.com → Template gallery) — bisa upload template kustom untuk seluruh organisasi (butuh akun Google Workspace).
📐

Mengatur Margin & Orientasi Halaman

▼
⬛ Microsoft Word
  1. 1Tab Layout → klik Margins → pilih preset (Normal, Narrow, Wide) atau Custom Margins…
  2. 2Di dialog Custom Margins: atur nilai Top, Bottom, Left, Right (dalam cm).
  3. 3Orientasi: tab Layout → Orientation → Portrait atau Landscape.
  4. 4Ukuran kertas: tab Layout → Size → pilih A4, Letter, dll.
✅Standar akademik UKWMS umumnya: margin Kiri 4 cm, Kanan/Atas/Bawah 3 cm. Simpan di template agar tidak perlu atur ulang setiap kali.
🔵 Google Docs
  1. 1Klik File → Page setup.
  2. 2Atur Paper size: A4, Letter, dll.
  3. 3Atur Orientation: Portrait / Landscape.
  4. 4Atur margin: Top, Bottom, Left, Right (dalam cm atau inci).
  5. 5Klik "Set as default" untuk menyimpan ke semua dokumen baru.
File → Page setup → Set as default

💡 Tips Formatting Profesional

🎯

Pakai Style, Bukan Format Manual

Selalu gunakan Heading Style alih-alih bold teks biasa. Style memungkinkan daftar isi otomatis dan perubahan global yang konsisten.

🧹

Clear Formatting Sebelum Paste

Setelah paste teks dari sumber lain, langsung tekan Ctrl+\ (Docs) atau Ctrl+Space (Word) untuk menghapus format bawaan teks tersebut.

📐

Konsistensi Margin Akademik

Standar Indonesia: margin kiri 4 cm (area jilid), dan atas/kanan/bawah 3 cm. Simpan sebagai template agar tidak lupa.

🖌

Format Painter untuk Konsistensi

Sudah format satu paragraf dengan sempurna? Gunakan Format Painter untuk menyalin formatnya ke semua paragraf serupa dalam hitungan detik.

🔤

Pilih Font yang Mudah Dibaca

Untuk dokumen akademik: gunakan Times New Roman 12pt atau Calibri 11pt untuk body, dan pertahankan konsisten di seluruh dokumen.

↔️

Justify untuk Laporan Formal

Alignment Justify (Ctrl+J) memberikan tampilan rapi di laporan dan skripsi karena teks rata kiri dan kanan secara bersamaan.
