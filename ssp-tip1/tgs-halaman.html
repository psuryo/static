📋

Skenario Tugas

Kamu diminta menyiapkan Laporan Kerja Praktek yang siap cetak dan dikumpulkan ke dosen pembimbing. Dokumen harus memiliki penomoran halaman yang berbeda antara bagian pendahuluan dan isi bab, heading yang konsisten dan sudah dikustomisasi, serta daftar isi yang dibuat secara otomatis oleh Word.

⚠️
Tugas ini hanya dikerjakan di Microsoft Word. Kerjakan secara berurutan: selesaikan Fase A dulu sebelum lanjut ke Fase B dan C, karena setiap fase bergantung pada hasil fase sebelumnya.
A
Fase A
Pengaturan Halaman & Nomor Halaman
📑 Struktur Dokumen

Dokumen terdiri dari 3 section dengan pengaturan nomor halaman yang berbeda. Buat terlebih dahulu struktur halaman berikut sebelum mengisi konten:

Section 1 — Pendahuluan
i

posisi tengah bawah

Section 2 — Awal Tiap Bab
1
Nomor arab

Section 3 — Isi Bab

💡
Penomoran angka arab menerus antara Section 2 dan Section 3 — halaman pertama bab = 1, halaman isi berikutnya = 2, 3, dst. Hanya posisi nomor yang berbeda, bukan angkanya.
🔧 Langkah Pengerjaan Fase A
A1

Siapkan Konten Dokumen (3 Bagian)

Persiapan

Buat dokumen Word baru, isi dengan struktur berikut. Gunakan teks fiktif yang masuk akal — konten yang sebenarnya akan diformat di Fase B:

  • Halaman Judul — nama, NIM, judul laporan, nama dosen, tahun (1 halaman)
  • Kata Pengantar — paragraf singkat 2–3 kalimat (1 halaman)
  • Daftar Isi — kosongkan dulu, akan diisi otomatis di Fase C (1 halaman)
  • BAB I — Pendahuluan — tiga sub-bab: 1.1 Latar Belakang, 1.2 Tujuan, 1.3 Manfaat (min. 2 halaman)
  • BAB II — Tinjauan Pustaka — dua sub-bab: 2.1 Teori Dasar, 2.2 Penelitian Terdahulu (min. 2 halaman)
  • BAB III — Penutup — dua sub-bab: 3.1 Kesimpulan, 3.2 Saran (min. 1 halaman)
⚠️
Belum perlu menggunakan Heading Style di langkah ini — cukup ketik teksnya. Heading Style diterapkan di Fase B setelah section break selesai dipasang.
A2

Atur Margin Dokumen

Layout
  • Tab Layout → Margins → Custom Margins…
  • Isi: Top = 3 cm, Bottom = 3 cm, Left = 4 cm, Right = 3 cm
  • Pastikan Apply to: Whole document → klik OK
Layout → Margins → Custom Margins…
A3

Pasang Section Break — Pisahkan 3 Bagian

Section Break

Section Break memisahkan dokumen agar tiap bagian bisa punya pengaturan nomor halaman yang berbeda:

  • Section Break 1: Letakkan kursor di akhir halaman Daftar Isi (sebelum BAB I). Tab Layout → Breaks → Next Page.
  • Section Break 2: Letakkan kursor di akhir halaman pertama BAB I (setelah judul BAB I dan sebelum paragraf isi). Pasang lagi Breaks → Next Page.
Layout → Breaks → Next Page
💡
Aktifkan Show Formatting Marks (Ctrl+Shift+8) untuk melihat tanda "Section Break (Next Page)" yang sudah terpasang. Ini membantu memastikan posisi break sudah tepat.
⚠️
Lakukan hal yang sama untuk setiap awal bab baru (BAB II, BAB III) — setiap halaman judul bab menjadi Section 2 tersendiri, dan isi babnya masuk Section 3.
A4

Nomor Halaman Section 1 — Romawi Tengah Bawah

Page Number
  • Klik di dalam halaman Section 1 (Halaman Judul / Kata Pengantar / Daftar Isi).
  • Tab Insert → Page Number → Bottom of Page → pilih Plain Number 2 (tengah).
  • Masih di mode edit footer, klik Page Number → Format Page Numbers…
  • Ubah Number format ke i, ii, iii, … dan pastikan Start at: i → klik OK.
Insert → Page Number → Bottom of Page → Plain Number 2
A5

Nomor Halaman Section 2 — Arab Tengah Bawah (Awal Bab)

Page Number
  • Double-click footer di halaman awal BAB I (Section 2) untuk masuk mode edit footer.
  • Di tab Header & Footer, klik Link to Previous untuk menonaktifkannya (tombol tidak lagi tampak tertekan/highlighted).
  • Hapus nomor halaman yang ada (jika otomatis terbawa dari Section 1).
  • Klik Insert → Page Number → Bottom of Page → Plain Number 2 (tengah).
  • Klik Format Page Numbers… → format 1, 2, 3, … → Start at: 1 → OK.
🔑
Nonaktifkan "Link to Previous" WAJIB dilakukan sebelum memasang nomor baru. Jika lupa, perubahan di Section 2 akan ikut mengubah footer Section 1 dan sebaliknya.
A6

Nomor Halaman Section 3 — Arab Kanan Atas (Isi Bab)

Page Number
  • Double-click area header di halaman isi bab pertama (Section 3).
  • Nonaktifkan Link to Previous di header (sama seperti langkah A5, tapi kali ini di header bukan footer).
  • Klik Insert → Page Number → Top of Page → Plain Number 3 (kanan).
  • Klik Format Page Numbers… → format 1, 2, 3, … → pilih "Continue from previous section" (agar angka menerus, bukan mulai dari 1 lagi) → OK.
  • Pastikan footer Section 3 tidak ada nomor halaman — hapus jika ada.
Insert → Page Number → Top of Page → Plain Number 3
💡
Ulangi langkah A5 dan A6 untuk setiap bab baru (BAB II, BAB III). Tiap awal bab punya Section 2-nya sendiri (tengah bawah), diikuti Section 3 untuk isinya (kanan atas).
A7

Verifikasi Hasil Pengaturan Halaman

Cek

Sebelum lanjut ke Fase B, pastikan semua kondisi berikut sudah benar:

  • Halaman Judul, Kata Pengantar, Daftar Isi → nomor romawi kecil di tengah bawah
  • Setiap halaman awal BAB → nomor arab di tengah bawah, dimulai dari angka 1
  • Setiap halaman isi bab → nomor arab di kanan atas, angka menerus
  • Tidak ada nomor halaman yang "bocor" ke section lain yang tidak seharusnya
⚠️
Gunakan Print Preview (Ctrl+P → pratinjau) untuk memverifikasi posisi nomor di semua halaman sebelum lanjut.
B
Fase B
Kustomisasi Style & Heading
🎨 Ketentuan Style yang Harus Diterapkan
Style Digunakan untuk Ketentuan Format Pratinjau
Heading 1 Judul BAB (BAB I, BAB II, dst.) Font: Times New Roman 14pt, Bold, warna #0D1B2A (navy). Border bawah merah tipis. Spasi sebelum: 24pt, sesudah: 12pt. BAB I Pendahuluan
Heading 2 Sub-bab (1.1, 1.2, 2.1, dst.) Font: Times New Roman 12pt, Bold, warna #2B579A (biru Word). Tidak ada border. Spasi sebelum: 12pt, sesudah: 6pt. 1.1 Latar Belakang
Heading 3 Sub-sub-bab (jika ada) Font: Times New Roman 12pt, Italic, warna #444444. Spasi sebelum: 6pt, sesudah: 4pt. 1.1.1 Konteks Masalah
Normal Semua paragraf isi Font: Times New Roman 12pt, warna hitam. Spasi 1,5. Alignment Justify. First line indent: 1,25 cm. Paragraf isi dokumen…
🔧 Langkah Pengerjaan Fase B
B1

Modifikasi Style Heading 1

Styles
  • Tab Home → di panel Styles, klik kanan Heading 1 → pilih Modify…
  • Ubah font ke Times New Roman, ukuran 14, centang Bold.
  • Klik tombol Format (pojok kiri bawah dialog) → Font… → ubah warna font ke #0D1B2A (pilih More Colors → masukkan kode hex).
  • Klik Format → Border… → pilih Bottom border, warna merah (#C0392B), lebar 1pt → OK.
  • Klik Format → Paragraph… → Spacing: Before: 24pt, After: 12pt → OK.
  • Centang "Automatically update" → klik OK.
Home → Styles → klik kanan Heading 1 → Modify…
B2

Modifikasi Style Heading 2 & Heading 3

Styles
  • Ulangi langkah B1 untuk Heading 2: font Times New Roman 12pt Bold, warna #2B579A, spasi Before 12pt After 6pt, tanpa border.
  • Ulangi untuk Heading 3: font Times New Roman 12pt Italic, warna #444444, spasi Before 6pt After 4pt.
B3

Modifikasi Style Normal (Body Text)

Styles
  • Klik kanan Normal di panel Styles → Modify…
  • Font: Times New Roman 12pt, warna hitam (Automatic).
  • Format → Paragraph…: Alignment Justified, Line spacing 1,5, Special indent First line 1,25 cm.
  • Centang "Automatically update" → OK.
B4

Terapkan Heading Style ke Seluruh Judul Bab & Sub-bab

Apply
  • Klik di baris teks "BAB I PENDAHULUAN" → klik Heading 1 di panel Styles. Ulangi untuk BAB II dan BAB III.
  • Klik di baris "1.1 Latar Belakang" → klik Heading 2. Ulangi untuk semua sub-bab.
  • Terapkan style Normal ke semua paragraf isi.
  • Verifikasi via View → Navigation Pane — semua judul harus tampil bersih di panel kiri.
💡
Shortcut: Ctrl+Alt+1 untuk Heading 1, Ctrl+Alt+2 untuk Heading 2. Jauh lebih cepat dari klik mouse.
B5

Gunakan Format Painter untuk Efisiensi

Format Painter
  • Jika ada teks yang formatnya sudah benar tapi belum diberi style, gunakan Format Painter untuk menyalin format secara cepat.
  • Seleksi teks sumber → klik ikon kuas di tab Home → klik/drag ke teks tujuan.
  • Double-click ikon Format Painter untuk menggunakannya ke banyak tempat sekaligus. Tekan Esc untuk berhenti.
C
Fase C
Daftar Isi Otomatis
🔧 Langkah Pengerjaan Fase C
C1

Sisipkan Daftar Isi Otomatis

Table of Contents
  • Klik di halaman Daftar Isi yang sudah disiapkan di Langkah A1 (halaman yang kosong).
  • Tab References → klik Table of Contents → pilih "Custom Table of Contents…"
  • Atur: Show levels: 2 (tampilkan Heading 1 dan Heading 2 saja). Tab leader: … (titik). Centang Show page numbers dan Right align page numbers.
  • Klik OK — daftar isi langsung terbentuk.
References → Table of Contents → Custom Table of Contents…
⚠️
Jika daftar isi kosong atau tidak ada isinya, berarti Heading Style belum diterapkan dengan benar di Fase B. Kembali ke Fase B dan pastikan semua judul bab menggunakan Heading 1/2.
C2

Kustomisasi Tampilan Daftar Isi

TOC Style
  • Di dialog Custom Table of Contents, klik Modify… untuk mengubah tampilan tiap level.
  • TOC 1 (Heading 1): font Times New Roman 12pt, Bold, warna navy.
  • TOC 2 (Heading 2): font Times New Roman 11pt, Regular, indent kiri 0,5 cm.
  • Klik OK untuk menerapkan.
C3

Update Daftar Isi & Verifikasi Akhir

Update
  • Klik di dalam daftar isi → klik Update Table yang muncul di atas → pilih "Update entire table".
  • Verifikasi: nomor halaman di daftar isi harus cocok dengan posisi bab yang sebenarnya di dokumen.
  • Periksa bahwa nomor halaman di daftar isi menunjukkan nomor angka arab (bukan romawi) sesuai penomoran section isi bab.
  • Lakukan Print Preview satu kali terakhir untuk memastikan keseluruhan tampilan dokumen sudah benar.
💡
Shortcut update daftar isi: klik daftar isi → tekan F9 → pilih Update entire table. Biasakan update daftar isi setiap kali menambah atau mengubah konten.
